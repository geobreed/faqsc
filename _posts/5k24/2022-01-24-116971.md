Selamat pagi, saya mau bertanya soal efaktur. Apakah bisa database efaktur disimpan di cloud yg lokasinya ada di luar negeri? Karena selama ini aplikasi nya harus terus hidup kan agar bisa menginput pk nya. apakah akan ada dampaknya jika servernya dalam virtual cloud? kl kaya gini harus backup db berkala ya mas/mba? krn fp yg baru diinput ga otomatis sinkron sm backupan datanya kan?

---

untuk db efaktur desktop kan bersifat offline. jika memang bisa disimpan di cloud, silahkan diupdate terus db nya, karena tidak update otomatis setelah perekaman data di db efakturnya. paling yang bisa pake cloud kemungkinannya efaktur host to host, tapi karena kita tidak punya modulnya, boleh arahkan ke kpp terkait efaktur host to host.
