terkait error e-Tax API 10025 : Dpokumen SPPB tidak ditemukan. WP sudah mencoba input sesuai format namun keluar error demikian, sudah coba ekspor dari web efaktur dan diimpor ke efaktur sesuai format juga tetap tidak bisa. Sebelumnya tadi pagi sudah menghubungi layanan telepon 1500200, disarankan untuk menunggu proses sinkronasi data. Terkait hal tsb, bagaimana ya mas mbak?

---

Silakan langsung dijawab mba, sesuai katalog error. Terkait error ETAX API 10025 Dokumen SPPB Tidak Ditemukan, ini dikarenakan Data SPPB belum ada/belum dipertukarkan dengan DJP. Solusinya: wp perlu konfirmasi ke DJBC terlebih dahulu. Apabila data DJBC sudah ada, maka konfirmasi ke DJP agar dilakukan pengecekan.
