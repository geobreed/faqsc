WP terkendala saat akan kirim spt masa ppn di web efaktur. muncul notif ETAX-WEB-00023: Rest Service error coba beberapa saat lagi. Kata WP sudah dicoba clear cache dan sudah terkendala sejak kemarin. apakah memang sedang ada informasi error atau ada arahan khusus?

---

Belum ada info error web base efaktur. Karena server ga ada info error, ada kemungkinan dari internet wpnya yang kurang stabil, diarahkan coba menggunakan provider lain yaa, C3L, mode penyamaran, browser lain, dan administrasikan ulang sertel
