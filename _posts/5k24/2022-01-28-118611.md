#42Q Saya mau tanya bu, kemarin DB efaktur saya kan sempat bermasalah ya tetapi sekarang sudah beres tetapi ada 2 faktur keluaran desember yang sudah dibatalkan pada saat itu tetapi tidak ada di administrasi faktur bu sehingga kalau saya posting SPM Desember nilainya jadi lebih besar (sama kyk faktur yang dibatalkan itu). Cara supaya bisa batalin faktur tsb gmn ya bu? Karen di administrasi faktur gaada dan ketika saya coba input ulang tulisannya no seri telah digunakan, itu bagaimana ya bu?

---

#42A: PM, sebenarnya tidak masalah karena pelaporan sekarang via web efaktur, namun jika WP menginginkan data FP yang sudah dibatalkan muncul di efaktur desktop, bisa cek DB Backup untuk ekspor-impor data FP tsb ke DB existing atau jika tidak ada backup bisa minta ke KPP terkait data FP keluaran tsb.
