#33Q: wp tidak bisa posting spt masa feb 22, sudah update aplikasi 3.2 saat posting muncul "SPT sudah pernah diposting, ingin melanjutkan posting sebelumnya? pilih No utk posting dari awal" kalau di klik no muncul notif posting sedang berjalan, tapi tidak bisa selesai sudah ditunggu. saat di lihat daftar posting tidak muncul yg masa tsb. Tapi kalo posting di web bisa, cuma krn buat keperluan administrasi jd ingin bisa juga posting di aplikasinya, apakah ada solusi mas/mba?

---

#33A: Ini WP kemungkinan memang sudah posting SPT nya Des, silakan ditunggu proses posting awalnya selesai dulu di efaktur desktopnya (takutnya size data nya cukup banyak jadi membutuhkan waktu). WP posting 2000 FP, mungkin memang memakan waktu lama. Bisa ditunggu atau jika ingin posting ulang silakan gunakan DB sebelum WP melakukan posting.
