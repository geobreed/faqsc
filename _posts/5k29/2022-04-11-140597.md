Ini sepertinya maksud WP data pada web-efaktur ada namun ketika disandingkan dengan aplikasi e-Faktur untuk FP 010.003-2248288684 tidak ada. WP input ulang muncul etax api 10004 nomor faktur sudah pernah digunakan. Sarananin WP untuk melakukan filter terlebih dahulu di aplikasi efaktur dia. Untuk filter jelaskan ini : 1. Klik menu Faktur -> Pajak Keluaran -> Administrasi Faktur. 2. Klik [F4]Filter. 3. Field, silakan pilih Nomer Faktur 4. Operator, silakan pilih "Like". 5. Nilai Pemban
