install aplikasi efaktur 3.2 tapi minta registrasi ulang

---

pastikan tidak salah memilih db, bisa cek status di eNofa solusi : 1. Jika status Registrasi eFaktur Desktop : Aktif Solusi : cari database terbaru yang sudah registrasi ulang. 2. Jika status Registrasi eFaktur Desktop : Aktif dan database terbaru tidak ketemu Solusi : Reset Aplikasi Client dan registrasi ulang. 3. Jika Status Registrasi eFaktur Desktop : Belum Registrasi Solusi : silakan lakukan Registrasi ulang
