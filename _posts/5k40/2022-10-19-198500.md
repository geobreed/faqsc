Apabila muncul keteranagn unhandled saat create csv ini dipastikan crystal report sama klik kanan run as administrator?

---

Boleh dipastikan notif lengkapnya, biasanya kalo notifnya muncul pas create csv memang masalah dengan crystal reportnya. Bisa dipastiin udah install crystall report di pajak.go.id ama klik kanan run as administrator
