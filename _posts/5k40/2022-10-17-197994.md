error unhandled exception espt PPh pasal 21

---

error unhandled exception: 1. solusi paling gampang dulu: RUN AS ADMINISTRATOR dengan klik kanan exenya; atau 2. Coba instal ulang crystal report; atau 3. Pakai cara teknis tapi agak ribet: a. Pertama cek Komputer/Laptop, menggunakan windows 32 bit atau windows 64 bit, karena caranya akan berbeda nantinya. b. Masuk ke Command Prompt. Caranya pada kolom Search programs and files ketik CMD kemudian tekan tombol ENTER, lalu muncul jendela Command Prompt atau tekan tombol CTRL + R ketik CMD enter/OK c. Untuk Windows 32 bit pada jendela command prompt ketik cd C:\Program Files\DJP\e-SPT Masa 21-26 2014\dll lalu tekan Enter. d. Untuk Windows 64 bit pada jendela command prompt ketik cd C:\Program Files(x86)\DJP\e-SPT Masa 21-26 2014\dll lalu tekan enter e. Setelah itu, ketik regsvr32 capicom.dll kemudian tekan ENTER Jika berhasil akan muncul notifikasi DllRegisterServer in capicom.dll succeeded. yang menandakan anda telah sukses melakukan register untuk capicom.dll f. Buat CSV kembali. atau 4. install ulang aplikasi (langkah terakhir kalau sudah mentok)
