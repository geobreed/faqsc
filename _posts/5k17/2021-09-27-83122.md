WP mengalami etax api 00023 kenapa ya? di katalog error tidak disebutkan.

---

ETAX API 00023 - tidak dapat mengambil data dari bea cukai. Pastikan format penginputan dokumen lain sudah sesuai. Jika sudah dipastikan sesuai tapi masih terkendala, disarankan menghubungi BC.
