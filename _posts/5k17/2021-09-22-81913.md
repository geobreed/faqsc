WP mengalami error di espt pph OP (runtime error! the application has requested the Runtime to terminate it in an unusual way)

---

1. boleh coba untuk ubah direktori penyimpanan file csvnya 2. kalo masih belum bisa, coba tutup aplikasi, buka lagi dengan klik kanan>run as administrator 3. kalo masih belum bisa, instal ulang aplikasi. Pastikan DB dibackup dulu 4. Sarankan untuk pake eform PDF Tapi kalo WPnya kekeuh pengen pake espt dan masih kendala, boleh arahin ke KPP aja
