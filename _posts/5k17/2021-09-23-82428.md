PT A Beli Software Warranty dari PT B, terutang PPh 23 kah?.

---

silahkan dipastikan apakah terdapat royalti, pengertian royalti ada di penjelasan UU PPH. selain itu ada juga terkait penegasan software berlisensi di S-654/PJ.03/2017. jika hanya penjualan software saja, maka tidak terdapat objek pemotongan pph pasal 23.
