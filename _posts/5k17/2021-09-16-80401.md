error efaktur desktop : 'jalankan aplikasi dari ETaxInvoice' . sudah disarankan untuk klik kanan pada EtaxInvoice yg berbentuk application masih gagal. ada saran selain c3l?

---

Coba cara ini 1. Matikan komputer nyalakan lagi, kemudian coba jalankan efaktur kembali 2. Coba buka aplikasi efaktur langsung dari folder efakturnya (bukan shortcut desktop), cari etaxinvoice dan buka dengan run as administrator 3. extrak aplikasi efaktur baru di folder lain, yg lama jangan dihapus, trus copas database dari aplikasi lama ke aplikasi baru, kemudian coba jalankan aplikasi baru
