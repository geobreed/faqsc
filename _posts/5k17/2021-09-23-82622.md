e spt tahunan badan, wp ada copy db spt untuk cabang, namun ketika dibuka di minta password db nya, padahal db kosongan

---

untuk spt tahunan harusnya yang lapor adalah npwp pusatnya. mungkin wp membuka file dbnya secara langsung bukan melalui e spt, kalau file db nya langsung diklik dua kali/dibuka memang ada muncul password. tapi kalau memang db kosongan harusnya ketika buka espt dan connect ke db akan muncul inputan data npwp dan profil lainnya jika db nya adalah db kosong
