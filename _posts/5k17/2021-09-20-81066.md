etax 10001 error database

---

[1] Tutup semua aplikasi e-faktur yang berjalan. Buka kembali file ETaxInvoice, dan pastikan hanya menekan satu kali sampai aplikasi e-faktur benar-benar terbuka. [2] Pastikan dalam folder db > ETaxInvoice terdapat folder log, seg0 dan tmp serta file db.lck, README_DO_NOT_TOUCH_FILE, dan service.properties [3] Pastikan folder db di aplikasi e-faktur berisi folder ETAXInvoice dengan isi folder sesuai pada nomor [2], pastikan pula tidak ada folder ETaxInvoice yang ditambahkan sendiri
