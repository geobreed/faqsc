WP simpan file pdf FP, tapi tidak ada bar code

---

harusnya ada bar code. coba langkah2 berikut: 1. pilih FP yg akan disave (approval sukses); 2. klik ikon PDF: 3. Pas keluar menu save in (defaultnya di mydocument) jangan langsung disimpan. file pdf tsb dicopy (Ctrl+C) pindahkan ke drive lain selain C: 4. Baru disave. 5. Cari di explorer, buka dan cek lagi apakah ada barcode
