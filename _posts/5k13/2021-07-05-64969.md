Kendala saat menginput bukti potong tidak final di Espt 21. Error inserting line no 2, 3 and 4. Exception: Command text was not set for the command object. olusinya gimana kak?

---

Kkemungkinan format NPWPnya tidak di custom menjadi 15 digit atau mungkin diubah menjadi text atau aposthrope. Format cell yg seharusnya dipilih wp untuk kolom NPWP : custom > pilih 0 > lalu ketik angka 0 sampai berjumlah 15 digit. Pastikan e-SPT PPh 21 sudah versi terbaru. Pastikan pengisian cells sudah benar dan sesuai.
