saya mau input ntp tp ga valid di PPN, kenapa ya??

---

efaktur desktop muncul error NTPN tidak valid, oba sarankan untuk pastiin KAP, KJS, Masa, Tahun nya dulu mas eko. Kalo udh sesuai bisa sarankan untuk dicek NTPN nya. selama sertel nya masih berlaku dan data pembayarannya masuk harusnya kan ga ada masalah e mas, sekarang jg ga ada info error buat server efaktur. Paling saranin WP input langsung di web-efaktur dulu, kalo aman ga masalah. Buat di desktop nya bisa dicoba secara berkala
