unhandled exception, belum kita ketahui pasti sebabnya apa. Jadi mari kita coba solusinya bersama2. 1. solusi paling gampang dulu: RUN AS ADMINISTRATOR dengan klik kanan exenya 2. setelah googling dengan keyword error yang panjang itu ternyata ada jawabannya tp ribet: a. Pertama cek Komputer/Laptop, menggunakan windows 32 bit atau windows 64 bit, karena caranya akan berbeda nantinya. b. Masuk ke Command Prompt. Caranya pada kolom Search programs and files ketik CMD kemudian tekan tombol
