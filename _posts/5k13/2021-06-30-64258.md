tidak bisa log in ke web efaktur karena tidak bisa input npwp, katanya sudah c3l dan ganti browser dan leptop masih sama saja

---

Kolom NPWP seharusnya terisi otomatis, sehingga PKP hanya perlu input password saja (= password eNofa) ketika akan login ke web efaktur. Jika NPWP dan Nama Perusahaan tidak muncul di menu login, hal ini disebabkan sertel belum terinstal di browser Silakan arahkan wp unduh ulang sertel yg masih aktif melalui eNofa, kemudian tanamkan ulang di browser. Sebaiknya di browser hanya ada 1 sertel saja yg tertanam. akses kembali laman web efaktur dengan sebelumnya lakukan c3l terlebih dahulu.
