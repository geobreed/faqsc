saat wp rest password djp online, muncul notif seperti ini, padahal efin sudah benar, bagaimana kak? reg 014 reset password tidak berhasil dilakukan

---

Berdasarkan katalog error untuk REG 014 reset password tidak berhasil dilakukan, disebabkan karena Update data ke DB gagal atau input password lamanya tidak sesuai. jika wp melakukan ubah kata sandi melalui menu profil djp online pastikan password lama yang diinput pada saat ubah kata sandi melalui menu profil djp online sesuai. sarankan lakukan c3l di browsernya. Jika masih gagal, bisa menggunakan menu "lupa kata sandi" pada tampilan awal djp online kembali dan coba di incognito/private brows
