Untuk pengisian impor FP Keluaran jika NPWP diawali angka 0 apakah harus diberikan tanda ' ?

---

pada tata cara pengisian di skema import seharusnya kolom NPWP langsung diisi nomor NPWPnya, tidak perlu tanda petik di depan. jika WP mengalami kendala demikian, infokan untuk memastikan cell kolom NPWP sudah menggunakan format number atau belum. untuk memastikan skema impornya benar, WP bisa mengekspor 1 faktur keluaran, kemudian isi data faktur keluaran yg akan diimpor di file hasil ekspor tsb.
