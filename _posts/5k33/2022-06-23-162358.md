#28A : Apakah notifikasi error yg muncul yakni " file help tidak ditemukan" ? Jika iya, Silakan dicek di folder efakturnya apakah sudah ada file help dengan nama “ EFaktur “ dan jenis file “ Compiled HTML Help file “ . Jika sudah ada , silakan klik 2x pd file tsb untuk membuka menu help efaktur . Jika tidak ada file tsb/file tsb tidak dapat dibuka kemungkinan file tsb corrupt , Silakan copy file tsb dari folder efaktur 3.2 lain jika ada/install lagi efaktur 3.2 dan copy file tsb dari installan
