Saat mau menjalankan efaktur terdapat notifikasi "java virtual machine starts failed", efakturnya tidak bisa berjalan, itu kenapa ya?

---

Kalau kucari ini ada beberapa solusi, coba cek java yg terinstall di pc wp, kalau sudah ada coba install ulang minimal versi 7. Kalau sudah install dan terupdate javanya dan masih keluar info error itu kemungkinan aplikasi efakturnya corrupt, coba sarankan wp install ulang aplikasinya trus db lamanya di masukan ke aplikasi yg baru diinstall td.
