Caranya pada kolom Search programs and files ketik CMD kemudian tekan tombol ENTER, lalu muncul jendela Command Prompt atau tekan tombol CTRL + R ketik CMD enter/OK c. Untuk Windows 32 bit pada jendela command prompt ketik cd C:\Program Files\DJP\e-SPT Masa 21-26 2014\dll lalu tekan Enter. d. Untuk Windows 64 bit
