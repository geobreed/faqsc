mas/mbak, impor csv pph 21 di espt muncul notif "format yg dipilih tidak sesuai", padahal sebelumnya pake file tsb bisa. Penyebabnya apa ya mas/mbak?

---

Ini kemungkinan besar disebabkan karena Template format CSV yg diimpor tidak sesuai dengan menu impor yang dipilih. Pastikan format file CSV impor yang dipilih sesuai dengan menu CSV Impor pada e-SPT PPh 21/26. Untuk format file CSV impor ini dapat dilihat pada Folder C:\Program Files\DJP\e-SPT Masa 21-26 2014\dokumentasi\csv format\contoh csv. Demikian pula jika salah menyimpan format CSV, maka data tidak bisa diterima. Data harus disimpan dalam format CSV (Comma Delimited).
