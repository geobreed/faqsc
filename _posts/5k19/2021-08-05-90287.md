Twitter: min data impor udah bs masuk tapi ada kode error gagal validasi terlampir berikut, ( ERREX0012 Data pemotong tidak ditemukan). Ini kenapa mas/mba?

---

Informasinya kemarin ada beberapa WP yg mengalami kendala yg sama. Data pemotong tidak ditemukan sepertinya hal ini karena dampak dari error SOA. Sistem jd tidak bisa membaca data NPWP, sehingga data pemotong tidak ditemukan. Belum ada informasi lebih lanjut terkait kendala tsb dari TIM terkait. Sarankan untuk WP coba kembali secara berkala yaa.
