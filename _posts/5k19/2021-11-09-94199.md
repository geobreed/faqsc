---

Kalau errornya pas cetak SPT arahkan wp untuk install crystal report ya kak, donlot dari pajak.go.id bagian aplikasi perpajakan, sesuaikan dengan bit komputernya. setelah install crystal report coba buka kembali dgn cara klik kanan pilih run as administrator. Kalau errornya pas buat file CSV 1. solusi paling gampang dulu: RUN AS ADMINISTRATOR dengan klik kanan exenya 2. setelah googling dengan keyword error yang panjang itu ternyata ada jawabannya tp ribet: a.
