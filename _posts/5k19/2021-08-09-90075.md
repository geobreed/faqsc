mas mba mau tanya, misal ada WP kemaren melakukan backup database efaktur. Terus hari ini melakukan prepopulated data, nah WP ini ingin menggunakan DB yg kemaren buat dimasukin ke folder efaktur skg. Sedangkan di DB kemaren data FP masukannya masih kosong ya. Apakah bisa mba/mas?

---

PM, DB hasil backup nya tidak bisa dibuka, WP menanyakan untuk PM yg terlanjur udah approval sukses di DB tersebut. Bisa pake DB dari backup lagi, nanti untuk PM nya bisa diinput manual dan upload lagi yg penting isiannya sama harusnya bisa, atau narik csv A2 dari web efaktur
