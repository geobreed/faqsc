cara patch patch espt pph 21 bagaimana yah ? karena saya install di direktory D:programfile/djp/ sedangkan waktu patch hanya ada opsi untuk folder C:/Program files

---

untuk patch e spt pph 21 memang langsung nembak ke folder C program files, karena defaultnya ketika wp install e spt ph 21 maka e sptnya masuk ke program files. kalau foldernya dipindahkan semua ke D, silakan pindahkan kembali ke C jika mau patch
