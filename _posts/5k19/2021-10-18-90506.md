mas mba kalau PIB masa september 2016 mau diupload reject etax api 50030 kenapa ya? udah dipandu agent sebelumnya untuk tatacara pengisiannya, tetep error gitu

---

Untuk ETAX API 50030 notifnya data tidak ditemukan. Pastikan memang tata cara penginputan di dokumen lain nya sudah sesuai. jika masih tidak ditemukan arahkan WP untuk konfirmasi ke BC terlebih dahulu melalui customer.beacukai.go.id atau telp. ke help desk Bravo Beacukai 1500225. terkait dengan SPT nya karena 2016 daluwarsa di 2021 september juga, PM kemngkinan tidak dapat lagi dikreditkan di SPT masa september tadi akrena sudah daluwarsa palagi kalau jadi LB/rugi SPT nya. daluwarsanya lebih
