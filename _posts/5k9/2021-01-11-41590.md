di web efaktur ini muncul etax-web-50008 itu kenapa ya mas mba?

---

pertama digali dulu status SPT nya? dokumen yang mau dilampirkan dalam bentuk PDF apa? Terus sampaikan bahwa ETAX-WEB 50008 ada beberapa kemungkinan: 1. file tidak sesuai ketentuan (format dan size lampiran) 2. terjadi kesalahan saat menyimpan file di server 3. terjadi kesalahan saat proses generate bps Solusi: Pastikan kembali format filenya pdf dan sizenya maksimal 5MB, untuk minimal tidak diatur. Lalu C3l, private window dkk + coba berkala.
