format impor dokumen lain pajak masukan, bisa didapat di mana?

---

Pada efaktur desktop 1. pilih menu help 2. klik dokumen help 3. buka Import Faktur/Dokumen Lain 4. buka Skema Import Faktur/Dokumen Lain 5. pilih Skema Import Dokumen Lain Bisa juga membuka file EFaktur.chm yang terdapat pada folder efaktur, lalu lanjutkan langkah 3 s.d. 5.
