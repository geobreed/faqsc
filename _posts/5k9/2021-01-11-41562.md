wp mengalami kendala seperti pertanyaan nomor 1 (ETAX-WEB 50008) saat upload lampiran pdf, kira2 kenapa ya?

---

Dari kemaren beberapa WP mengalami hal yang sama, tapi belum ada pernyataan secara resmi dari tim terkait ETAX-WEB 50008 ada beberapa kemungkinan: 1. file tidak sesuai ketentuan (format dan size lampiran) 2. terjadi kesalahan saat menyimpan file di server 3. terjadi kesalahan saat proses generate bps Pastikan size file nggak terlalu besar, Untuk saat ini, silakan sarankan juga c3l/incognito/browser lain, coba berkala
