Etax api 50031 sppbmcp. Penginputan sudah sesuai. Apakah konfirmasi BC?

---

Apabila statusnya masih reject, silakan coba dihapus dulu, keluar dari efaktur, jalankan menggunakan run as administrator, kemudian coba input kembali dengan format AWB#NTPN (coba input dengan semua huruf kapital). Untuk kolom tanggal diisi dengan tanggal yg tercantum di SSP dengan format dd-mm-yyyy Pastikan jg jenis transaksi, dokumen, detail transaksi dan dokumen transaksi benar ya
