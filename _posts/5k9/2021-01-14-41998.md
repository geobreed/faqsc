Error ketika import bukti potong A1 pada aplikasi e-spt 21

---

Periksa kembali penginputan skema import bukti potong A1 kemungkinan ada kesalahan pada nomor bukti potong. Bisa menggunakan contoh Excel CSV bukti potong A1 pada folder dokumentasi - csv format - contoh csv - 1721_bp_A1. Atau Input manual di aplikasi e-spt nya
