Mau cetak induk di e-faktur, etax 50005, sudah coba cara yg disampaikan agent sblmnya ttp blm bisa

---

Untuk ETAX-50005, kemungkinan penyebabnya hanya terkait isian SPT yang belum lengkap/tersimpan, data profil PKP yg belum lengkap/update serta pilihan lokasi penyimpanan file yg tertuju di (C:). Mohon pastikan kembali instruksi yg sebelumnya telah disampaikan, terutama terkait lokasi penyimpanan file cetakan/PDFnya, sangat Taxmin sarankan untuk kembali coba pilih lokasi penyimpanan di desktop atau selain (C:)
