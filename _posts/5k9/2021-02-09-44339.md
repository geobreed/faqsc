Unhandled exception pada eSPT PPh 21?

---

memang unhandled exception unik, kita ga tau sebabnya apa. Jadi mari kita coba solusinya bersama2. 1. solusi paling gampang dulu: RUN AS ADMINISTRATOR dengan klik kanan exenya 2. setelah googling dengan keyword error yang panjang itu ternyata ada jawabannya tp ribet: a. Pertama cek Komputer/Laptop, menggunakan windows 32 bit atau windows 64 bit, karena caranya akan berbeda nantinya. b. Masuk ke Command Prompt. ( Jawaban tidak bisa diinput secara lengkap akibat keterbatasan karakter)
