wp saat mau lapor spt ppn masa desember muncul error etax-web-50008: spt service-00046: terjadi kesalahan saat upload lampiran Ini gimana ya ?

---

untuk katalog error yang etax web kan kita belum ada update ya tapi kalau dicari di https://ortax.org/ortax/?mod=forum&page=show&idtopik=88462 ada yang bisa lapor tanpa upload lampiran. paling digali dulu aja "status SPT-nya nihil, KB, atau LB?" ETAX-WEB 50008 ada beberapa kemungkinan 1. file tidak sesuai ketentuan (format dan size lampiran) 2. terjadi kesalahan saat menyimpan file di server 3. terjadi kesalahan saat proses generate bps Pastiin filenya pdf dan ukurannya ya maks 5mb.
