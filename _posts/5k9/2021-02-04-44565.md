saya upload impor e bupot, namun muncul informasi error " status kesalahan upload : (object object)" itu kenapa ya pak?

---

nfo terakhir ebupot error object object. 1. cek isian file excelnya (terutama format hrs sesuai) 2. coba donlod ulang file xls skema impor yg terbaru, gunakan file yg baru (isian dr file lama dicopas). coba upload ulang file tsb.. 3. Jika 2 cara tsb msh muncul notif yg sama ketika impor, wp diarahkan untuk bawa file skema impor tsb ke kpp untuk dicek
