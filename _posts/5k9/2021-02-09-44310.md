WP upload ebupot lewat csv, kemuadian keluar pesan kesan upload [object object]. Solusinya bagaimana ya Mas/Mbak?

---

Upload ebupot lewat csv itu maksudnya gimana ya ndra (?) Biasanya kalo object objet muncul ketika upload skema impor ebupot. 1. Cek isian file excelnya (terutama format harus sesuai) 2. Coba download ulang file xls skema impor yg terbaru/ gunakan file yg baru bisa minta ke KPP kalau ngga ada di menu ebupotnya (isian dari file lama dicopas). coba upload ulang file tsb. 3.
