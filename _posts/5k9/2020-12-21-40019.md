etax service 10002, kenapa dan bagaimana solusinya?

---

Biasanya krn koneksi ke database terputus/tidak berhasil. Pastikan PC Server dan Client terkoneksi dengan baik, apabila menggunakan network DB, pastikan DB di server sudah berjalan. Tutup aplikasi e-faktur di client, koneksikan kembali ke server dan kemu- dian ulangi proses (atau jika pakai lokal DB, cukup restart aplikasi e-faktur)
