Untuk pph 23 buat liat data2 yg lama, jadi harus liat di e spt nyaa , karena bukti potongnya ada di espt, untuk data yg dilapor sebelum pake ebupot. Tetapi tidak terbaca database nya. Itu bagaimna yaa?

---

Pada saat login espt 23 muncul keterangan user name anda salah, silahkan coba kembali. Secara default user dan passwor espt 23 administrator 123, kalo memang gak bisa kemungkinan user name dan passwordnya udah pernah drubah di menu sett password username. Kalo memang ingin membuka data di db tersebut dan lupa username yg pernah diganti bisa coba hubungi AR nya dulu utk mereset username password nya dgn kirim database
