Saya mau bertanya mengenai cara instal Aplikasi PPh 21 di PC. Barusan saya sudah Instal Crystal Report Runtime untuk Windows 64 bit terlebih dahulu, tapi tidak ada hasil apa-apa di PC nya. Lalu pada saat mau instal pph pasal 21-26 versi 2400 pilih yang Single Instaler tapi tidak muncul apa-apa. Bagaimana ya caranya?

---

Jika WP merupakan pengguna baru eSPT Masa PPh 21/26 maka WP dapat install dengan file Single Installer Aplikasi eSPT Masa PPh Pasal 21/26 Versi 2.4.0.0 yang telah disediakan di laman pajak.go.id tanpa perlu install versi sebelumnya. Jika sudah berhasil unduh single installer maka di dalamnya akan ada folder debug>e-SPT package>klik “Next”. Jika instalasi sudah selesai, maka akan muncul tampilan installation complete. Kalau lokasi installnya gak diubah WP bisa buka melalui file: C>Program Files>D
