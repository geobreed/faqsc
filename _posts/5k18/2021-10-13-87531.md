Tahun buku antara di profil web efaktur dengan aplikasi efaktur dekstop berbeda. harusnya jan-des, namun di web efaktur okt-sep?

---

data tahun buku di profil pkp web efaktur ternyata salah. sebenarnya data yang muncul di profil pkp web efaktur merupakan data yang terdapat dalam sistem djp ya, jadi jika terdapat perbedaan data, silahkan hubungi kpp terdaftar untuk konfirmasi data yg tertera di sistem ya.
