apakah web efaktur sedang main tenis? Saat submit spt tidak valid, udah berbagai clear cache tetep muncul itu

---

kemaren ada beberapa yang nanya gini, buat hari ini belum ada info error. bisa dicoba cara ini dulu sis SPT Tidak Valid di Web Efaktur: 1. Melakukan Clear Cache 2. Menghapus Semua Sertel di Browser 3. Unduh ulang sertel melalui enofa Online 4. Tutup Browser 5. Impor Sertel kembali ke Browser yang digunakan 6. Akses kembali menu efaktur Web. 7. Sarankan kepada WP agar tidak menyimpan (bookmark) halaman eFaktur Web kalau masih tetep error, coba dihapus dan posting ulang ya
