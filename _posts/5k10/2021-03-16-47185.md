database e-spt pph 21 saya hilang min krna terkena banjir jadi file nya terformat saat perbaikan pc, apa ada solusi min? Saya mau lapor pph 21 min mohon bantuannya. selain input ulang apakah ada solusi lain yg bs disampaikan y mas/mbak?

---

Apabila memiliki backup db tsb, maka nanti bisa koneksi database ke db backup-an tadi. Namun jika tidak punya, ketika hilang maka tidak ada solusi selain input ulang/baru, karena database e-spt sifatnya offline, beda sama e-faktur yg bisa minta ke KPP.
