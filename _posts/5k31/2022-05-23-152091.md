#31Q: Apakah ada info error terkait ebupot unifikasi mas/mba? Pada saat unduh file bukpotnya kenapa tidak muncul hasil downloadnya, padahal status downloadnya sukses ?

---

Sementara sih belum ada info error ya mba maya, Kalo dia mau unduh bupot di menu dashboard, boleh dipastiin ini dia iconnya masih ajukan unduh bupot apa sudah berubah icon unduh bupot. Kalo masih muter2, pada saat ajukan unduh bupot memang ada proses request ke server DJP agar dapat mengunduh bupot. Infonya akan langsung berubah dan akan muncul tombol aksi Unduh Bupot pada SPT. Saat ini belum ada info error lebih lanjut mba, kalo klik ajukan unduh itu malah muncul blank putih, standar bisa c3l d
