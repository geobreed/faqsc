etax 30005 saat impor data pk dari kpp

---

Silakan menambahkan kolom KODE_DOKUMEN_PENDUKUNG di baris FK terlebih dahulu pada file CSV yang akan diimpor. Bila hasil permintaan data ke KPP, File CSV tersebut terbentuk di folder yang sama dengan file zip permintaan data (nama file CSV sama dengan nama file zip; file tersebut terbentuk setelah file zip diimpor ke database Wajib Pajak).
