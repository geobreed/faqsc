#27Q wp dapat notif etaxservice-20017 client tidak terdaftar. disarankan oleh agent sebelumnya utk reset apk client. sudah coba copas folder db dan reset, fpk malah hilang. bagaiman ya ini mas mba? https://twitter.com/tinysugaarplum/status/1511208871112880128

---

#27A: Halo indira, ETAXSERVICE-20017 : Client Tidak terdaftar Menandakan bahwa Aplikasi pernah di reset, sehingga database tersebut tidak dapat lagi menjalankan uploader atau tidak dapat mengupload faktur. Jika sudah registrasi aplikasi baru dan WP masih ada db lama yg gak rusak/ada folder backup dbnya, buat data FPKnya nanti bisa pake mekanisme ekspor impor dari db lama, cuman kalo ngga ada backup db, bisa arahkan WP buat minta data FPK yg udah approval sukses ke KPP ya in
