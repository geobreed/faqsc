#18Q https://twitter.com/creampuffchoco/status/1511273050716934146 : Izin tanya mas mba "Aplikasi efakturnya sudah ditutup kak. Lalu muncul seperti ini. File db nya error Solusinya bagaimana ya?"

---

#18A: pastikan db yg dicopy gak corrupt. misal ada file yg hilang atau pada saat copy db dari yg lama ke yg baru. kalo errornya pas update 3.2, pastikan cara/urutan update nya sesuai. 1. Rename Folder eFaktur lama, tambahkan "_old" misalnya. 2. Download dan extract eFaktur, pastikan folder terpisah dan tidak menimpali folder eFaktur lama. Berikan nama tambahan, misalnya "_v3.2" . 3. Copy folder "db" pada efaktur lama, dan pindahkan ke folder efaktur baru yang telah ter-extract. 4. Klik mem_
