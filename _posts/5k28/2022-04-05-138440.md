Tarif HS code antara pmk 142/pmk.010/2017 dan pmk 26/pmk.010/2022 atas soya lecithin berbeda. apakah pmk 26/2022 mengubah pmk 147? apakah hs code soya lecithin mengikuti pmk 147 atau pmk 26?

---

silahkan sesuaikan dengan pmk 142/pmk.010/2017. pada lampirannya tertulis untuk yang dibebaskan adalah soya lecithin dgn hs code ex Ex 2923.20.10. saat ini pmk 26//pmk.010/2022 tidak mengubah pmk 142/pmk.010/2017, jika dilihat topik pmk nya juga berbeda.
