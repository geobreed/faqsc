ketika submit stuck sesuai SS, dan untuk TA reporting kenapa error ya?

---

Coba sambil dikonfirmasi mas. Untuk error stuck tersebut apakah terjadi pada file eform? Jika iya, pastikan Browser IE bisa mengakses internet baik proxy atau direct3. Pada menu: IE> Internet Option> Advanced> pastikan SSL dan TSL tercentang semua. Untuk error TA ereporting, mohon sampaikan dengan rinci error yg terjadi. Error terjadi ketika apa dan bagaimana notifikasinya.
