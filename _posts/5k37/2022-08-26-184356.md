#37Q awalnya wp mau posting masa PPN 07 tetapi tidak muncul dalam list aplikasi SPT, lalu mau posting ulang tulisannya sudah terposting. kira2 solusinya seperti apa ya?

---

#37A: Diarahkan untuk klik perbaharui tampilan dahulu namun kata WP masih tetap tidak muncul dan error, coba close aplikasi dan jalankan dengan run as administrator juga masih error. Pertama mungkin bisa dijelasin kondisi sekarang tidak perlu posting di aplikasi efaktur dekstopnya karena untuk posting hingga pelaporan sudah dilakukan melalui web-efaktur. Kedua kalo mau boleh arahin WP-nya untuk cek kapan dia posting SPT masa tersebut, kemudian cek folder backup, cari db terbaru sebelum dia melakukan posting SPT, kemudian coba gunakan db tersebut
