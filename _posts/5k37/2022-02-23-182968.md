saya mau tnya untuk import CSV PPN tetapi kode error nya 30005. ============================= ini kenapa ya Mas/Mbak?

---

ini impor faktur biasa mas dan keterangannya Error objek mapping? Kalo iya bisa dijelaskan sesuai katalog, penyebabnya karena Format isian data csv yang diimpor tidak sesuai dengan skema impor dari DJP. Bisa saranin WP buat Periksa kembali file csv dan pastikan isian data telah sesuai dengan skema impor DJP. Untuk langkah2 dan skemanya bisa minta WP buat mastiin di help efaktur bagian import faktur/dokumen lain. Atau kalo masih pake format lama sebelum 3.1 yg nomor dokumen pendukung, silakan WP bisa mendapatkan skema impor yg terbaru.
