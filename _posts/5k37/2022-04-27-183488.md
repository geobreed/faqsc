apakah bisa proses faktur menggunakan pjap tetapi lapornya menggunakan web efaktur?

---

yg dimaksud wp: Buat faktur Menggunakan pjap lapor menggunakan web efaktur. Karna kalo lapor melalui web faktur kan posting spt dulu dan itu mengambil dari data sistem djp kita ga tau juga sistemnya di pjap gimana, apakah datanya bisa sinkron ke web faktur atau tidak. bisa konfirmasi ke pjapnya. Soalnya kalo posting spt di web efaktur mengambil data faktur dari sistem djp. Kita ga tau dari data pjapnya itu konek ke sistem djp atau engga.
