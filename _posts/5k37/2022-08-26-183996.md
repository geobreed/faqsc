ETAX 40003, sudah regis ulang dan download aplikasi terbaru tp masih muncul ket error itu saat upload FP ada saran lain kah kak?

---

Error ETAX-40003 : Error di service Session ETAXSERVICE-20017 : Client Tidak terdaftar Menandakan bahwa Aplikasi pernah di reset, sehingga database tersebut tidak dapat lagi menjalankan uploader atau tidak dapat mengupload faktur. Solusi : 1. Jika status Registrasi eFaktur Desktop : Aktif Solusi : cari database terbaru yang sudah registrasi ulang. 2. Jika status Registrasi eFaktur Desktop : Aktif dan database terbaru tidak ketemu Solusi : Reset Aplikasi Client dan registrasi ulang. 3. Jika Status Registrasi eFaktur Desktop : Belum Registrasi Solusi : silakan lakukan Registrasi ulang NB : Untuk datanya dapat dipindahkan dengan cara ekspor-impor. Jika Anda Kesulitan, silakan minta bantuan IT Support di KPP
