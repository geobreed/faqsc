saya mau terbitkan faktur pajak 07 untuk kawasan bebas dengan melampirkan PPBJ. di reject dengan alasan, nomor peraturan seharusnya sudah Nomor 41 tahun 2021. sdangkan di efaktur saya, muncul peraturannya masih nomor 10 tahun 2012. saya sudah update menjadi efaktur 3.1. bagaimana cara agar nomor aturannya menjadi 41 tahun 2021 ya?

---

Mas ini udah update di sinkronisasi cap atau belum ya? Kalo emang di sinkronisasi cap belum muncul yg no 41, wp bisa bisa input manual di bagian referensi yaa Pastikan langkah sinkronisasi kode cap nya udah bener dulu ya mas, harusnya udah ada di efaktur 3.1 1. Pilih Menu Referensi => Sinkron Kode Cap 2. 2. Setelah memilih Menu di atas akan muncul Form “Sedang Memproses” (Menunggu Koneksi antara WP dan DJP) 3. Jika WP belum menjalankan Uploder maka akan muncul Form “Login User PKP”. Ketika Muncul Form “Login User PKP” => Masukkan Captcha dan Password e-Nofa (Password Aktivasi) => Tekan Tombol Submit 4. 4. Setelah itu akan muncul Form “Input Captcha” untuk mengambil data Kode Cap dari server DJP. Jika sebelumnya sudah melakukan Start Uploader maka akan langsung diarahkan ke Form “Input Captcha”, lalu masukkan captcha yang muncul dengan benar (Bersifat Case Sensitive – huruf besar dan kecil harus sesuai) => Tekan Tombol Validate (sebanyak 2 kali) 5. Akan muncul Daftar Kode Cap Faktur Pajak 6. Melakukan Uji Coba Pembuatan Faktur dengan Kode Transaksi 07 / 08 di menu input Daftar Faktur Pajak Keluaran
