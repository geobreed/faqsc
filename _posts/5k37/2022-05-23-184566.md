Saya sebelumnya posting di aplikasi efaktur desktop kemudian posting di web efaktur, namun di lampiran induk, angka ppn keluaran berbeda antara di desktop dan di web efaktur. Setelah saya telusuri ternyata selisihnya ada pada jumlah ppn keluaran faktur kode 05. Padahal di lampiran A2 baik di aplikasi desktop maupun di web efaktur angkanya sama. akan tetapi di lampiran induk di aplikasi efaktur desktop utk kode faktur 05 tidak ketarik di jumlah angka ppn keluaran. Kiri dari aplikasi efaktur desktop dan kanan dari web efaktur. Terlihat di sini angka ppn keluarannya berbeda Setelah dicek berikut jumlah lampiran a2 di web efaktur: dan berikut lampiran a2 di aplikasi efaktur desktop:. angka di lampiran A2 sama namun kenapa di induk bisa berbeda ya?

---

sebenernya ini beberapa WP sebelumnya juga menginformasikan case yang sama, yaitu untuk faktur pajak dengan kode 05 ternyata jumlahnya ngga terakumulasi di induk e-faktur.. Namun belum ada info juga dari pihak terkait (termasuk info error). Jadi saranku ngga ada salahnya utk menyarankan coba posting ulang dulu lagi, keluar aplikasi, buka lagi dg klik kanan run as administator, baru buka lagi spt utk masa tsb.. Kemudian cek lagi di A2 sama induk, kalo di induk dekstop tetep ngga muncul/terakumulasi jumlahnya, kita jawabnya selama di web-base sudah muncul/terakumulasi dengan benar sesuai nominal seharusnya, ini ngga masalah karena pelaporan spt masa ppn kan lewat web base yaa, artinya kalau webbase sudah benar dan sesuai, harusnya ngga ada masalah.. Jangan lupa mohon maaf dulu atas ketidaknyamanannya..
