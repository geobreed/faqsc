Apakah ada solusi lain selain solusi dibawah ini troubleshoot seputar eForm: 1. Jika ini adalah eForm pdf, gunakan Adobe Reader versi 20 ke atas. 2. Browser IE bisa mengakses internet baik proxy atau direct3. Pada menu: IE> Internet Option> Advanced> pastikan SSL dan TSL tercentang semua. 4. Ekstensi file pdf adalah pdf bukan PDF. 5. Pada nama file pdf, tidak boleh ada karakter titik (.) lebih dari satu. 6. Ukuran file pdf tidak boleh melebihi 40MB7. Tidak ada isian yang masih merah. 8. Pada kolom Keterangan isian Harta, jika kosong isikan "-". 9. Time zone perangkat adalah GMT+7 Jakarta. 10.
