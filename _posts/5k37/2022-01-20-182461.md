PPS> WP mau submit PPS muncul kendala spti di photo, ada info server error kah, mba mas?

---

Silakan coba ikuti langkah-langkah berikut: 1. Save file e-form SPPH terlebih dahulu, kemudian tutup. 2. Buka Internet Explorer, kilk Options 3. Di bagian Network and Internet, cari Internet Options. Setelah terbuka, klik Advanced di pojok kanan atas. 4. Di bagian Setting, scroll ke bawah cari bagian Security (gambar gembok kuning). Kemudian, centang semua SSL dan TLS. 5. Buka kembali file eform SPPH(.pdf). Coba submit form SPPH kembali. 6. Jika belum berhasil, restart perangkat yg digunakan 7. Pastikan koneksi internet stabil / coba gunakan koneksi internet lain dan coba submit e-form SPPH kembali
