ETAX-40003 : ETAXSERVICE-20017 : Client Tidak terdaftar Menandakan bahwa Aplikasi pernah di reset, sehingga database tersebut tidak dapat lagi menjalankan uploader atau tidak dapat mengupload faktur. Solusi : 1. Jika status Registrasi eFaktur Desktop : Aktif Solusi : cari database terbaru yang sudah registrasi ulang. 2. Jika status Registrasi eFaktur Desktop : Aktif dan database terbaru tidak ketemu Solusi : Reset Aplikasi Client dan registrasi ulang. 3. Jika Status Registrasi eFaktur Desktop : Belum Registrasi Solusi : silakan lakukan Registrasi
