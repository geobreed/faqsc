siang min, pelaporan ppn skrg kan menggunakan web efaktur, apakah spt di app efaktur masih perlu di posting? Thx min

---

pelaporan SPT masa PPN melalui web based. aplikasi desktop digunakan untuk membuat dan menginput faktur pajak. Wp tidak perlu posting spt di efaktur desktop sepanjang tidak diperlukan. Wp bisa juga posting spt di efaktur misalnya untuk keperluan pembandingan pengisian spt efaktur dengan web based atau mau melihat lampiran2 spt.
