error saat prepop, apakah ada info maintanance?

---

Belum ada info error Solusi etax 10004 dan etax 40001 sbb: 1. Cek apakah Internet aktif dan lancar 2. Jika menggunakan proxy maka di aplikasi efaktur juga harus setting proxy 3. Cek apakah efaktur diblok oleh antivirus atau firewall 4. Download ulang sertel lalu input ulang di efaktur 5. Setelah input ulang, tutup aplikasi efaktur. Buka kembali aplikasi dengan klik kanan lalu pilih "run as administrator" 6. Jalankan uploader dan ulangi proses get data di menu prepopulated
