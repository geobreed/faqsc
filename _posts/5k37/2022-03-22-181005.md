Mohon pencerahan mengenai input data dokumen lain ekspor dimana pada bulan desember 2021 lalu, saya menghubungi kring pajak untuk pengaduan dokumen lain ekspor REJECT dengan keterangan ETAX API 50041 ( NPWP PEMILIK TIDAK SESUAI DENGAN DATA BEA CUKAI), namun setelah saya konfirmasi ke bea cukai untuk data nya tidak ada masalah. Dari keterangan bea cukai saya harus konfirmasi ke DJP. Dari pihak DJP mengatakan untuk input nama penerima eksportir tidak dipermasalahkan dan bisa diubah ubah, yang sebelumnya saya input a/n HAKAMATA WORKS CO.,LTD selaku penerima eksportir dan pada proses upload dengan keterangan Reject. Lalu saya coba memasukkan data sesuai dengan PEB dengan nama HAKAMATA UTAMA bisa upload succes. Jadi bulan selanjutnya Januari 2022, saya input data dengan nama PT.HAKAMATA UTAMA upload succes. Akan tetapi di bulan februari ini saya mencoba untuk memasukkan data dengan nama penerima eksport HAKAMATA WORKS CO.,LTD , bisa succes. Yang mau saya tanyakan : 1. Apakah ada perubahan sistem atau upgrade sistem di DJP yang sebelumnya data dengan nama penerima ekspor reject jadi succes? 2. Untuk bulan februari 2022 ini saya input data harus sesuai yang mana? 3. Jika untuk ubah data di bulan desember 2021 dan juga Januari 2022 bisa, apa yang selanjutnya harus saya lakukan?

---

Untuk pengisian PEB di dokumen lain PK, silakan disesuaikan dengan keadaan sebenarnya mas, sesuai PEB nya dan sesuai pasal 5 ayat 1 PER 16/2021. Secara aplikasi, PEB itu kan data dari BC mas, dan itu datanya dipertukarkan dengan kita, sehingga butuh waktu untuk pertukaran datanya tsb. Ketika data tsb belum dipertukarkan, maka ketika wp upload, kemungkinan reject. Saat ini secara sistem efaktur saat menginput PEB salah satunya NPWP pemilik barang akan divalidasi, sehingga apabila PEB tidak tercantum NPWP PKP sebagai pemilik barang atau NPWPnya berbeda, maka akan gagal validasi. Pastikan npwp peb a.n. pemilik si wp. untuk penginputan di dok lain dgn no peb#no aju, atau bisa langsung ambil di prepop Untuk yang ada perubahan ini coba mas diminta detil perubahannya seperti apa. jika ada kesalahan untuk ubah dokumen lain selain nomor dokumen dan npwp sebenernya kan bisa ya, apabila tidak berhasil ubah dpp dengan klik ubah, memang disarankan untuk ubah jenis transaksi ke penyerahan dalam negeri, DPP dan PPN di 0 kan. Setelah itu, input ulang PEB dengan format yang baru. penginputan nomor dokumennya no PEB#no AJU. pastikan no AJU yang diinput tanpa tanda hubung.
