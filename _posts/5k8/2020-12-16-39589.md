Efaktur web base lg error kah? Permasalahan WP tidak muncul identitas WP di halaman login web efaktur, cuma blok merah saja

---

1. Pastikan tidak salah buka browser, biasanya WP install sertel di chrome tapi dia bukanya di mozilla, jika sudah dipastikan terinstall coba pake incognito/private mode 2. Bisa juga karena sertelnya sudah nggak berlaku, pastikan masih aktif di enofa 3. Download ulang sertel, pastikan penamaan sertelnya tidak ada penambahan apapun saat didownload 4. Install sertel di OS,, 5. Hapus semua sertel pkp yang ada di browser, kemudian upload lagi, kemudian coba pakai Incognito/private mode
