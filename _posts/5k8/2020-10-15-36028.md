client dan server selalu minta upload sertel tiap kali mau upload

---

sebenarnya kan aplikasi client itu pake db aplikasi server ya. kalo terjadi seperti itu, sarankan untuk aplikasi server diinput lagi sertel nya di referensi>administrasi sertifikat. jadi gak cukup masukin sertel ketika mau upload tok. kemudian kalo udah diinput ulang sertel, start db sebagai server lagi di menu administrasi db.
