WP mengalami eror saat upload PDF di web base efaktur. etaxweb 50008

---

ETAX-WEB 50008 ada beberapa kemungkinan: 1. file tidak sesuai ketentuan (format dan size lampiran) 2. terjadi kesalahan saat menyimpan file di server 3. terjadi kesalahan saat proses generate bps Pastikan size file nggak terlalu besar, sarankan juga c3l/incognito/browser lain, coba berkala
