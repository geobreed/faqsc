Petunjuk pengisian csv untuk peb bisa dilihat di efaktur dekstop: 1. pilih menu help 2. klik dokumen help 3. buka Import Faktur/Dokumen Lain 4. buka Skema Import Faktur/Dokumen Lain 5. pilih Import Dokumen Lain
