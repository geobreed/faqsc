kalo user name dan password e-spt PPh badan rupiah 2010 versi 1.2 apa ya? Saya input username: administrator dan pass: 123 tidak bisa ya.

---

Defaultnya itu username administrator password 123. Pastikan dilu apakah wp pernah mengubah password nya pada menu utility set username-password. Kalo pernah diubah, yang mengetahui password hanya orang yang mengubah, mau tidak mau buat db baru kalo tidak tau username dan passwordnya.
