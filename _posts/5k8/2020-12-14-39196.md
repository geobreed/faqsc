WP mengalami kendala eSPT PPh OP sperti terlampir Ini karena WP blm install access database engine kan ya? lalu bisa juga kan ya solusi selain install acces db tsb, aku infokan bisa juga pake eform?

---

betul, salah satu solusi untuk mengatasi munculnya error microsoft ace oledb 12.0 is not registered on the local machine adalah dengan menginstall driver access database engine (nanti silakan WP cari sendiri, karena kita tidak menyediakan). Di samping itu, karena WP adalah OP yang melakukan pekerjaan bebas, maka boleh sarankan juga untuk lapor SPT Tahunan menggunakan e-Form.
