Cetak bukti potong espt 21 saat dia centang pilihan cetak langsung muncul jendela save file, ini default nya begini atau tergantung settingan printer default nya dia ya?

---

Ganti setting default printer,kalau default nya yang Microsoft print to PDF, dia langsung ngecreate file pdf sehingga diminta memasukkan alamat save filenya. Kalau default nya diganti ke printer yang di punya, akan langsung cetak.
