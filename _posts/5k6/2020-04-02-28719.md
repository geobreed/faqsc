Apakah yg dimaksud upload csv efaktur di djponline? bisa dicoba pas bikin file csvnya di aplikasi efakturnya, induk bagian VI (kelengkapan SPT) untuk lampiran AB sampai B3 tercentang. Kalo masih gabisa bisa arahkan WP create ulang CSV, file CSVnya jangan dibuka, dan gunakan private browsing atau clear cache. Kalau masih tetap gak bisa, bisa disarankan lapor via pos/ekspedisi ke kpp atau asp.
