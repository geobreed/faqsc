SPT masa. Nama file PDF tidak sesuai?

---

klo dilihat dari notif errornya kan simpel ya sebenernya, nama PDF nya tidak sesuai. berarti kita jawab aja sesuai errornya itu, dilengkapi dengan contoh Pemberian nama file lampiran SPT selain untuk SPT PPh Badan (1771) dan SPT Tahunan OP (1770) yang berekstensi pdf harus sama dengan nama file csv, kasih contoh aja.
