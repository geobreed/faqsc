database error ketika koneksi

---

utk keterangan error seperti itu, udah bener format filenya. Cuma itu harus dicek lagi tata cara setting utk database nya melalui ODBC apakah sudah benar. Lalu, apakah dia ingin buat database baru atau copy database dr aplikasi di komputer lain? Takutnya klo copy database, sebenernya yg dicopy bukan database aslinya (bisa jadi database asli di virtual store), pastikan lagi aja dan bisa dikasih tau utk ini. Dan yg paling penting, adalah tata cara setting database melalui odbc jangan sampe salah
