tidak bisa buat file csv (spt badan) muncul “file not found zlib.dll”

---

error ini karena file zlib.dll di aplikasi WP crash dan butuh recovery. Silahkan cari file zlib.dll (bisa dari internet atau di komputer), terus diletakkan lagi di direktorinya. a. Untuk Windows 32 bit Buka direktori C:\Windows\System32\1771 2010 RP, Copy file zlib.dll kemudian paste ke direktori C:\Program Files\DJP\eSPT 1771 2010 setelah itu coba jalankan lagi e-SPT nya dan create CSV.
