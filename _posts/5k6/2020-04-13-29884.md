database hilang

---

Bisa beberapa cara, jika masih memungkinkan untuk mengakses efaktur dan bisa membukanya, bisa dgn ekspor impor datanya. Jika db yg hilang tidak keseluruhan, atau ada error ketika efaktur dibuka, bisa menggunakan file backup terakhir yg ada di folder backup utk extract dan pakai db yg terakhir terecord itu. Jika benar-benar hilang semua, harus minta database ke KPP dan krn KPP sdg tidak ada layanan tatap muka, bisa dengan mekanisme sesuai SE13 saja
