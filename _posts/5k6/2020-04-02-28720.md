WP bertanya apakah spt badan rupiah sudah kompatibel dengan windows 10, kata wp data tidak bisa diupdate, dijawab bagaimana ya?

---

Muncul notifikasi tidak updatenya bagaimana? Apabila wp ingin menginstall epst 1771 silakan bisa diklik link berikut https://pajak.go.id/id/e-spt-tahunan-pph-badan-full-patch. Nanti download semua file zipnya, lalu ekstrak file zip yg nomor 1. Setelah diekstrak akan ada folder yang di dalamnya ada installer dan file text tentang cara instalasi. Bisa diarahkan untuk mengikti cara-cara yang ada tata cara tadi, ESPT badan 1771 bisa digunakan di windows 10.
