Format penamaan lampiran file pdf untuk kelengkapan laporan SPT Tahunan Badan yang benar gimana kah?

---

untuk format penamaan lampiran pdf pastikan sesuai dengan format nama sesuai petunjuk di efilingnya. Selain nama file harus sama dengan nama file CSV, WP harus menambahkan kode lampiran dibelakangnya. Misalnya Lampiran Laporkan Keuangan, nama file lampiran PDF-nya harus (nama file CSV)LK atau 1234567896160000112201800F1132140111LK
