terkait upload impor ebupot pph 23, status seperti di gambar dan jumlah baris saya ada 9, kenapa munculnya hanya 1 ya? nanti kalau berhasil generate berantakan gak ya file saya?

---

Untuk upload file excel impor bukti potong di ebupot lama (bukan unifikasi) pastikan menggunakan format excel terbaru yg dapat diunduh di petunjuk pengisian. Sebelum upload file excel pastikan juga format impor bupot sudah divalidasi. Jumlah baris yg terbaca di aplikasi ebupot seharusnya sesuai dengan jumlah baris yg direkam di file excel, silakan dicek kembali hal-hal ini.
