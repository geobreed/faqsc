#53Q Ganjar WP tidak dapat log in ke aplikasi eSPT PPh 21 dengan notifikasi eror NPWP harus 15 digit. Padalah WP sudah pernah berhasil log in dan lapor SPT untuk bulan-bulan sebelumnya. (Seharusnya tinggal log in dengan username dan password kan?). Ini kira-kira kenapa ya Mas/Mba? Apakah ada kesalahan di database atau minta WP install ulang aplikasinya? Notifikasi awal ketika membuka aplikasi adalah Silakan mengisi NPWP terlebih dahulu.

---

#53A: PM. kalo diminta NPWP biasanya itu di aplikasi baru install atau DB baru. dipastikan yg dipake aplikasi dan db yg biasa dia pake. input NPWP dicoba di espt ku bisa. saranin agennya pake db baru aja, terus pastikan di referensi kode KPP dia ada.
