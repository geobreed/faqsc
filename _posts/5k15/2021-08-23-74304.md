#115Q WP upload csv espt PPh Pasal 4 (2), "proses tidak berhasil -1"

---

1. Silakan dicek dan perhatikan kembali untuk terkait jenis file CSV, penamaan file csv, dan size file CSV (jangan sampai sizenya berukuran 1KB karena itu ada indikasi CSV tidak berhasil digenerate. 2. Clear cache dan cookies pada browser. 3. Gunakan New Private Window (Mozilla Firefox)/New Incognito Window (Chrome) untuk melaporkan SPT nya. 4. Coba ganti browser. 5. Coba menggunakan perangkat yang lainnya. Kendala error “Proses tidak berhasil (-1)” saat lapor SPT, Ibu juga dapat me
