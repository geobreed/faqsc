WP mengalami error etax 10002 tidak dapat mengambil data pada saat prepopulated. Kondisi WP membuka data base dg laptop ari rumah karena WFH. Sertel masih berlaku dan sudah dicoba import ulang sertel tetap sama, ini bagaimana ya mas/mba?

---

sesuai katalog error itu disebabkan karena koneksi antara server dan client terputus (network database). Pastikan pc server dan client terkoneksi dengan baik. Tutup aplikasi di client, koneksi kembali ke server, terus coba proses prepopulated lagi. Kita saranin juga pastikan koneksi lncar dan stabil, tutup dulu efakturnya dan coba gunakan koneksi lain, matikan antivirus di latopnya juga ya... Opsinya coba berkala dan terakhir kalau masih gak bisa input manual
