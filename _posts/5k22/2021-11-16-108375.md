mas/mbak wp nanya > saya baru install, aplikasi pph 21 versi 2.4.0 cristal runtime 32bit & 64bit. masalahnya : tidak bisa cetak lampiran. kalau spt induk nya, bisa dicetak. dimana masalahnya ya?

---

Wp diarahkan untuk install crystal report sesuai dengan bit di komputer nya. Wp merupakan pengguna baru espt pph 21, silakan unduh dan install yg single installer. kalau WP unduh single installer dari pajak.go.id memang isinya hanya folder debug saja, langkah selanjutnya di folder debug --> e-SPT package --> klik “Next”. Jika instalasi sudah selesai, maka akan muncul tampilan installation complete. Pada saat cetak lampiran muncul error application has stopped working dan muncul keterangan
