Saya mau tanya efaktur 3.1 kenapa saat buka di komputer windows 7 etaxinvoicenya tidak bisa dibuka, tapi pada saat buka di windows 10 tidak ada masalah, padahal versi sebelumnya yg 3.0 masih bisa buka di windows 7. Apakah versi windows pengaruh jg di efaktur versi 3.1?

---

Pm. Untuk saat ini sudah dikonfirmasi ke tim helpdesk efaktur dan tidak ada masalah harusnya bisa berjalan normal di windows 7 ataupun 10. Yg bisa disarankan dulu untuk saat ini: - klik kanan run as administrator - pastikan system requirement selain OS windowsnya di PC yg windows 7 tadi terpenuhi (bisa cek menu help di efaktur desktopnya yg bisa dibuka tadi) - kalau tidak ada urgensi di pc windows 7 tadi, silakan wp pakai dulu aja aplikasi efaktur yg bisa berjalan normal di PC windows 10nya
