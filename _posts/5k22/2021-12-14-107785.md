Tutorial update patch efaktur2.2 menjadi 3.0 bagaimana ya Pak, mohon petunjuknya?

---

"1. Download file patch yang tersedia di 5 operating sistem di efaktur.pajak.go.id. 2. Extract file tersebut. Hasil extra : ada 3 file. Etaxinvoicemain.exe, etaxinvoice.Exe dan etaxinvoiceupd 3. Sebelumnya back up dulu file yang ada (bisa di rename). Lalu copas 3 file tersebut dan akan mereplace file existing. 4. Jalankan Etaxinvoice.exe. Akan muncul update : butuh waktu beberapa waktu. 5. Kalo berhasil di aplikasi akan ada menu prepop faktur pajak masukan dan dokumen impor pib."
