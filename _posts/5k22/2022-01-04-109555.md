# 74 Q mau menanyakan terkait efaktur 3.1 hari ini saya update kok tidak bisa ya? Keterangannya 'Database tidak bisa di akses, pastikan Aplikasi E-Faktur tidak berjalan' sedangkan dikomputer tsb tidak ada aplikasi efaktur yg sedang kami buka kalau seperti ini hrs bagaimana ya?

---

#74A proses update 3.1nya belum berhasil, coba update lagi ulang. pastikan pas copy paste folder dbnya ke efaktur 3.1, aplikasi efakturnya dalam keadaan tertutup. abis copy paste DBnya, jalankan dulu ETaxInvoiceUpd, abis itu baru jalankan ETaxInvoice. kalo masih ga bisa, kemungkinan DBnya rusak, kalo kayak gini ambil folder DB dari backup terakhir di efaktur yang lama, folder DB ini nanti yang di copy paste ke efaktur 3.1
