Di dalam web-efaktur PT kami bagian administrasi SPT,tiba2 tiap bulan nya ada pembetulan 1 secara otomatis tanpa dibuat. itu error dari web faktur nya atau gimana ya ? Kondisi di web efaktur wp saat ini muncul postingan pembetulan-1 di setiap masa pajak sejak tahun 2020 tetapi hanya posting saja tidak sampai lapor, namun wp merasa tidak pernah posting pembetulan-1 tersebut. Apakah memang terdapat error tersebut? terkait kondisi WP tersebut diarahkan bagaimana ya

---

belum ada info error web efaktur bim. Jika memang wp merasa tidak ada pihak lain yang mengakses web efaktur wp tersebut, log out web efaktur, c3l browser dan coba login kembali... Jika masih muncul pembetulan 1 padahal wp gak merasa posting spt, arahkan wp untuk hapus saja postingan spt pembetulan 1 nya bim
