saya mencoba untuk impor data excel di aplikasi e bupot pph 21, tetapi muncul peringatan gagal dengan indikasi seperti ini?

---

Jika memperhatikan kode error yg muncul di ebuni IP ketika impor, ini fokusnya adalah ke kolom NPWP/NIK, kode objek pajak, penghasilan bruto, DPP, dst. minta wp untuk memastikan kembali pengisian di sheet tsb. Kemudian, sarankan juga wp untuk mengunduh ulang skema impor PPh 21/26 IP (ada di kotak petunjuk pengisian web ebupot IP), agar bisa dilakukan pengisian ulang untuk baris/bupot yg error ketika validasi. Pastikan wp tidak mengubah2 format cells yg telah ada.
