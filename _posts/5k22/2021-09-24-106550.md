---

Jika database wp dirasa penuh, wp bisa pake db baru untuk efaktur Bisa dipindahin dgn mekanisme ekspor impor Misal ada db A dan db B, nanti dari db A bisa dipindah ke db B dgn mekanisme ekspor impor wp create database baru terlebih dahulu kemudian data lama baru dipindah. Buat DB baru bisa melalui Menu file > administrasi DB
