Mas/Mba WP input PIB di efaktur, reject ket ETAX-API-50030: data tdk ditemukan, silahkan cek ke portal BC, sudah cek ke BC infonya data pembayaran sudah sesuai sudah input Impor PIB : No.pendaftaran#ntpn. Solusi lainnya apa ya mas/mba?

---

Untuk ETAX API 50030 kan notifnya data tidak ditemukan. Pastikan memang tata cara penginputan di dokumen lain nya sudah sesuai. untuk tatacara penginputan nomor dokumen tergantung dokumen impor WP, bisa diliat cara ini ya *Piloting*, user efaktur 3.0 Impor PIB (bc2.0), bc 2.8: No.pendaftaran#ntpn Sptnp, spktnp: 6 digit setelah sptnp/spktnp- #ntpn Sppbmcp: no.AWB#ntpn Jkpln: ntpn#kodekpp / pbk#kodekpp bc 2.3 masuk ke b3 bc 2.5 , bc4.1 input di 2b induk Kode kpp adalah kode kpp admin
