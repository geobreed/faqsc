Siang, ini jadi keluar error setelah memasukkan password untuk start uploader. 1. Sertifikat masih berlaku 2. Sertifikat sudah di import ulang dari enofa 3. Antivirus / Firewall sudah di matikan. 4. Bulan kemarin masih bisa untuk membuat faktur keluaran (saat versi sudah update). etax-40003 etax service-20017

---

Error ETAX-40003 : Error di service Session ETAXSERVICE-20017 : Client Tidak terdaftar Menandakan bahwa Aplikasi pernah di reset, sehingga database tersebut tidak dapat lagi menjalankan uploader atau tidak dapat mengupload faktur. Solusi : 1. Jika status Registrasi eFaktur Desktop : Aktif Solusi : cari database terbaru yang sudah registrasi ulang. 2. Jika status Registrasi eFaktur Desktop : Aktif dan database terbaru tidak ketemu Solusi : Reset Aplikasi Client dan registrasi ulang. 3. Jika Status Registrasi eFaktur Desktop : Belum Registrasi Solusi : silakan lakukan Registrasi ulang NB : Untuk datanya dapat dipindahkan dengan cara ekspor-impor.
