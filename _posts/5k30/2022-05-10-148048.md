saya Billy dari PT Jasa Web Indonesia saat ini kami memiliki aplikasi ERP yang dipakai klien kami bernama jasaerp.com kami menohon bagaimana prosedur untuk menjadj mitra yang mendapatkan akses API PPn PPh ebupot dan PPh23 e-1721 agar program kami yang digunakan invoicing dan salary pegawainya dapat terintegrasi dengan pajak? yg saya butuhkan API adalah Application Programming Interface dimana koneksi antara program cloud kami dengan database pajak bukan Angka Pengenal Impor (API) tolong dibantu

---

Ini kayanya WP ingin jadi PJAP kali ya, tapi dia pengennya aplikasinya dia terhubung sama db pajak. Sambil dipastikan ke WPnya apa bener yg dimaksud WP adalah PJAP, jelasin prosedur sesuai yg ada di PER-11/2019 sttd PER-10/2020 aja sa. Pastikan dia memenuhi persyaratan di pasal 2 dan 3. Permohonannya sesuai sama lampiran I dan dokumennya di lampiran I huruf B.
