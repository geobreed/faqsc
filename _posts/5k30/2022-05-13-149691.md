#58Q : WP upload efaktur muncul pesan “ etax 20002: Nomor faktur sudah terdaftar”. WP mau buat FP pengganti, udh masukin kode fp pengganti (mekanisme impor) tp gabisa kenapa ya Mas/Mbak?

---

#58A: Sepertinya ini FP-K Normal sudah diupload tapi tidak ada di Database WP nya (bisa jadi karena hilang/multiple DB). Pertama arahkan pakai menu filter di daftar PK untuk mencari FP Normal dengan nomor tsb. Jika tidak ada, silakan minta data FP-K ke KPP terlebih dahulu (Pasal 35 PER 03/2022). Kalau WP input FP Pengganti, tapi FP Normal tidak ada di database muncul error (gambar).
