WP mau menjalankan database server efaktur muncul error E-tax:10006 sudah disampaikan sesuai katalog tetap tidak bisa gimana ya kak?

---

penyebabnya biasanya karena Port yang digunakan untuk koneksi database belum dibuka. solusinya Pastikan port yang dipakai untuk koneksi database telah dibuka. jika masih ada trouble coba cek kembali konfigurasi server client nya. coba sarankan wp back up db dulu, konfigurasikan ulang saja sesuai menu help.
