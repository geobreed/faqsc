bagaimana cara impor excel ebupot unifikasi utk pph 23? apakah bisa impor excel, dan menggunakannya excel xls?

---

Buka tab Pajak Penghasilan->pilih menu Impor Data PPh->buka Petunjuk Pengisian->klik "disini" untuk mengunduh file excel impor-> buka dan isi file excel sesuai petunjuk pengisian di dalamnya (ada segitiga merah kecil)->nama file bisa ikuti petunjuk pengisian yg di menu Impor Data PPh tadi->jika sudah selesai silakan diimpor ke ebuni file excelnya
