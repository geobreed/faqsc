Boleh digali dulu. Muncul error tsb saat wp klik apa? Apakah wp menggunakan network database client server untuk efakturnya? Hari ini belum ada informasi terkait kendala error tsb. Kalau berdasarkan katalog error efaktur error tsb penyebabnya koneksi antara server dan client terputus (network database). Solusinya pastikan pc server dan client terkoneksi dengan baik. Tutup aplikasi efaktur client, koneksikan kembali ke server dan kemudian ulangi proses.
