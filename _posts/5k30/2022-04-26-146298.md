saat melakukan import file CSV ke Apps Efaktur, Kendala nya adalah penggunaan value VAT pada saat penghitungan transaksi tidak sesuai, yang seharusnya penghitungan VAT transaksi menggunakan 11%, Tetapi apps Efaktur menggunakan VAT 10%. Namun jika dinput secara manual value VAT nya sesuai yaitu menggunakan 11%. Sebelumnya masalah ini tidak pernah terjadi, penggunaan VAT selalu sesuai.

Jika WP memerlukan format impor selain cara diatas, wp juga bisa meminta ke kpp langsung ya.
