#110Q : siang pak, ini saat akan isi bag B untuk data peg yg di bawah PTKP pph 21 kok tidak bisa ya?

---

#1A By pm. Error unhandled exception. memang unhandled exception ini banyak kemungkinannya. 1. solusi paling gampang dulu: RUN AS ADMINISTRATOR dengan klik kanan exenya; atau 2. Coba instal ulang crystal report; atau 3. Pakai cara teknis tapi agak ribet: a. Pertama cek Komputer/Laptop, menggunakan windows 32 bit atau windows 64 bit, karena caranya akan berbeda nantinya. b. Masuk ke Command Prompt. Caranya pada kolom Search programs and files ketik CMD kemudian tekan tombol ENTE
