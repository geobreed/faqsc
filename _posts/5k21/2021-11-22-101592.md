---

ini benar pembetulan ya mas ? Boleh disarankan beberapa langkah - coba buat pembetulan di espt 1771 versi 2009, jadi silakan edit di versi 2009 namun create csv ttp menggunakan versi 2010. - ada juga kasus yg tidak bisa isi lampiran V spt 1771 krna sudah input spt 2018. Jadi spt 2018 dihapus baru isi yg 2017. Lalu untuk spt 2018 ini bisa di back up dh impor ekspor beberapa data atau silakan back up database dg copy ke folder lain dlu. - jika wp berkenan, bisa buat spt pembetulan menggunakan
