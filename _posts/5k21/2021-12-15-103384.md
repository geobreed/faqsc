WP tidak bisa buka prepopulated, muncul error seperti digambar. Sudah disarankan untuk dicoba rekam manual, tetap tidak bisa, pas mau upload gk ada verifikasi captcha dan notifikasi error tetap sama.

---

Untuk Error ETAX-10004, silakan klik get data kembali untuk mengulangi proses request ke server DJP karena kemungkinan koneksi terputus. Coba juga untuk stop uploader dan nyalakan kembali. Coba rekam manual 1. Pastikan koneksi internet stabil. 2. Coba matikan Antivirus dan/atau Firewall sebelum buka aplikasi efaktur kembali. 3. Jika menggunakan proxy, pastikan aplikasi efaktur juga sudah dilakukan setting proxy. 4. Pastikan sertel masih berlaku, silakan coba untuk unduh kembali sertel
