#5Q: wp lapor spt ppn melalui web faktur terdapat error etax web 00023-rest service error coba beberapa saat lagi ketika submit, sudah c3l, apa ada info error?

---

belum ada info error efaktur, etax web 00023 kemungkinan karena: 1. dari internet wpnya yang kurang stabil, diarahkan coba menggunakan provider lain, dicoba menggunakan private window dan posting ulang SPT. 2. server efaktur yang sedang sibuk / maintenance
