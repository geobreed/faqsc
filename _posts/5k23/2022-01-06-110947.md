#87Q Wp ada error etax-api 10025= sppb tidak di temukan tapi pas upload lagi jadi etax api 10025=npwp penjual tidak valid dan ada 1 lagi etax eror etax-api 10027=documen sppn sudah pernah di gunakan padahal belum pernah. Ini bagaimana ya mas/mba?

---

#87A By pm. Error etax api 10025 ternyata untuk bc 2.7. Terkait bc 2.7 implementasinya di efaktur 3.1 masih menunggu ND dari PP1 dan TPB. Error etax api 10027 untuk bc 4.0, pastikan sppb tidak pernah diinput untuk fp lain.
