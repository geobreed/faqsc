etax api 10031 pada saat input PPBJ di efaktur

---

Apakah ETAX-API-10031 nomor dokumen PPBJ tidak ditemukan? Jika iya, silakan: -pastikan inputan nomor PPBJ di eFaktur desktop sudah sesuai (tidak memasukkan 3 digit pertama, karena itu Kode PPBJ, bukan Nomor PPBJ). -Lalu tanggal FP pastikan tidak boleh mendahului tanggal PPBJ. -Arahkan PKP Penjual/Penerbit FP bisa memastikan data PPBJ melalui sistem INSW secara mandiri.
