WP mau instal eSPT PPh 21 versi 2.4. ada tampilan sepeti ini, kenapa ya? padahal sudah instal crystal report jg.

---

unhandled exception ini belum kita ketahui pasti sebabnya apa mba. bisa dicoba cara ini dulu soalnya beberapa WP coba ini berhasil . Error unhandled exception: 1. solusi paling gampang dulu: RUN AS ADMINISTRATOR dengan klik kanan exenya; atau 2. Coba instal ulang crystal report; atau 3. a. Pertama cek Komputer/Laptop, menggunakan windows 32 bit atau windows 64 bit, karena caranya akan berbeda nantinya. b. Masuk ke Command Prompt. Caranya pada kolom Search programs and files ketik CMD kemud
