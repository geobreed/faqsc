izin tanya, apakah e-form PPH Badan bisa digunakan untuk tahun buku april-maret?

---

seharusnya tidak masalah, setelah digali WP eror server ketika submit. Troubleshoot seputar eForm: 1. Jika ini adalah eForm pdf, gunakan Adobe Reader versi 20 ke atas dan disarankan 32bit 2. Browser IE bisa mengakses internet baik proxy atau direct3. Pada menu: IE> Internet Option> Advanced> pastikan SSL dan TSL tercentang semua. 4. Ekstensi file pdf adalah pdf bukan PDF. 5. Pada nama file pdf, tidak boleh ada karakter titik (.) lebih dari satu. 6. Ukuran file pdf tidak boleh melebihi 40MB
