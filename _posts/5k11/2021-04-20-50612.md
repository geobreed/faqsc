selamat pagi admin, saya mau bertanya terkait pelaporan e-bupot menggunakan excel di djponline. 1 file excel terdiri dari beberapa NTPN, saya bigung di bagian input bukti setor apakah harus input NTPN per invoice? total pajak ada 43 baris soalnya. jika hanya input 1 NTPN nanti di bagian panel perekaman bukti penyetoran tidak muncul ID, NO, Nomor Bukti, Jenis Pajak, Jenis Setoran, Tahun, Masa, Jumlah dan Tanggal Setor. Apakah ada solusi? Maaf karena saya baru mencoba menggunakan excel agar lebih

---

Jika yg dimaksud adalah input bukti setor di panel perekaman bukti penyetoran, maka harus semua ntpn yg diinput. Jika pembuatan bukpot melalui mekanisme impor, tidak perlu input ntpn di excel, input ntpn jika mendapatkan fasilitas DTP saja.
