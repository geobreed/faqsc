Saya baru saja mendapatkan telepon elektronik dari 1500200 hari ini jam 17.11 sore. Disebutkan bahwa data pelaporan Pajak saya tidak terlapor di system perpajakan selama 5 tahun. Saat saya cek di website DJP online, saya melihat riwayat pelaporan saya hanya 2019 dan 2020. Saya menduga data saya telah hilang di system anda. Saya memiliki bukti jika saya sudah melaporkan SPT secara rutin sejak tahun 2015 melalui e-spt (tahun sebelumnya manual kertas) seperti yang terlampir. Mohon dibantu investiga

---

Kan kring pajak memang ada layanan outbound yg mengimbau wp lapor spt berdasarkan history di sistem kita ya. Harusnya sih udah ambil data terupdate. Kalo memang wp yakin sudah lapor, coba arahkan hubungi kpp untuk konfirmasi dulu ya sambil menunjukkan bukti2 pelaporan ke kpp. Kalau memang ternyata sistem pelaporannya yg nggak sinkron dengan masterfile kita, biar nanti kpp coba bantu tindak lanjut pake lasis kemungkinan
