mas/mba kalo muncul error "Anda menggunakan espt versi lama" di eform xfdl ini solusinya gimana ya? disarankan unduh ulang versi xfdl nya apa diarahkan untuk pake yg versi pdf aja?

---

1. coba download ulang dari DJP Online. 2. Apabila sudah download eForm terbaru masih saja muncul error "Anda Menggunakan SPT versi Lama, Silahkan Unduh (Download) Versi Terbaru" silahkan anda coba submit ulang hingga berhasil. Karna ada yang sampai dua atau tiga kali submit baru berhasil. 3. Apabila masih belum berhasil, boleh anda coba unggah file lampiran PDF minimal dua file lampiran. 4. Kalau tetap gabisa, bisa disarankan menggunakan eform PDF atau menggunakan espt tahunan badan
