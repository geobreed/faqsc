Saya ingin tanya apakah penyebab aplikasi espt PPH 21 yang tidak bisa melakukan cetak Induk dan Formulir ? Padahal saya sudah install CRRuntime Aplikasi apa saja yang harus di install sebelum aplikasi eSPT PPh 21 ?

---

Aplikasi yg harus ada sblm install espt 21, bisa dilihat dari file bawaan espt 21, ada DotNetFX40Client, WindowsInstaller3_1, crystal report. Tapi kalau lihat notifikasinya masih krn crytal reports, bisa krn proses installnya blm berhasil. Kasih link ini saja: https://pajak.go.id/id/aplikasi-pajak/crystal-report-runtime
