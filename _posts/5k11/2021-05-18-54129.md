lupa password admin utama

---

Tahapan reset password admin utama ada 3 yaitu : 1. Permintaan Reset Password dari Aplikasi eFaktur 2. Reset Aplikasi eFaktur dari website eNofa Online (https://efaktur.pajak.go.id) menu Reset Aplikasi Client 3. Registrasi Ulang dan Ubah Password Admin Utama
