espt badan 2010 tidak bisa dibuka, error 1706 no valid source could be found itu gmn ya?

---

Dari berbagai sumber dapat disimpulkan bahwa ini bukan masalah di aplikasi (dalam kasus ini: e-SPT) nya, melainkan pada Windows Installer WP. Sebaiknya arahkan WP untuk install ulang espt 1771 2010 (di perangkat yg berbeda). Dan sarankan WP untuk konsul lebih lanjut dg IT mereka, terkait kendala sistem operasi yg mereka gunakan tsb. Atau coba reboot lagi komputernya, lalu uninstall terus install lagi di komputer yg sama. Tapi imho agar lebih clear, sebaiknya wp coba di perangkat yg berbeda.
