notif error tersebut biasanya muncul kalo format penamaan file nya ga sesuai atau wp masih pake format Excel yg lama mas. Arahkan wp nya download format Excel yg baru di bagian petunjuk sebelah kiri ya, sama pastiin format penamaan file nya gini: AAAAAAAAAAAAAAA_BBCC_DDDD_EE_FF.xls A : 15 digit (NPWP), B : 2 digit (Masa Pajak Awal), C : 2 digit (Masa Pajak Akhir), D : 4 digit (Tahun Pajak), E : 2 digit (Kode Pelaporan Realisasi) F : 2 digit (Kode Pembetulan Ke-) Kalo untuk pph 21 DTP, y
