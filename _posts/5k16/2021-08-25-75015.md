#75Q: Saya mau tanya, apakah sedang ada masalah dengan server efaktur ? Karena saya upload faktur dari tadi reject terus min sudah coba close dan buka lagi, tetep gabisa upload

---

#75A By pm. Dari grup helpdesk efaktur tadi jam 14.26 ada laporan etax api 00001 server sedang sibuk, pihak IT bilang "Mohon maaf atas ketidaknyamanannya, saat ini sedang dicek performa database e-Faktur nya" Kemungkinan emang lg kendala di server efaktur Tapi coba sampaikan gini aja, Dicoba unduh dan impor ulang sertelnya ke aplikasi efaktur, ganti koneksi internet, stop dan start uploader ulang Kalau masih gabisa dicoba kembali berkala ya
