saya ada install aplikasi e spt ppn 1111dm, namun ketika saya ingin simpan bagian induknya tidak bisa. muncul tampilan eror dengan keternagn 'Failed (Insert_Induk) Message : Data Mismatch In Criteria Expression'

---

silahkan bisa diarahkan untuk coba solusi berikut ini ya. Tutup aplikasi eSPT-nya. Masuk Control Panel pilih Clock, Language, and Region. Klik Region and Language.Cek pada bagian Formats - Long Date, pastikan dan ubahkan dalam format "ddMMMMYYY". Klik Apply dan OK. Apabila cara diatas seketika belum berhasil, boleh dicoba juga ganti format longdate ke dMMMMyyyy dulu, jika masih tidak bisa coba silahkan restart PC/laptop
