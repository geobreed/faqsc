format penamaan file laporan realisasi

---

Pastikan format penamaan file sebagai berikut: AAAAAAAAAAAAAAA_BBCC_DDDD_EE_FF.xls A : 15 digit (NPWP), B : 2 digit (Masa Pajak Awal), C : 2 digit (Masa Pajak Akhir), D : 4 digit (Tahun Pajak), E : 2 digit (Kode Pelaporan Realisasi) F : 2 digit (Kode Pembetulan Ke-)
