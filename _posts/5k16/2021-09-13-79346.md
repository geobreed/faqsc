aku abis install eSPT PPh 21, tapi kenapa file yg dotnet sama acces database engine nya gada ga? Trus itu perlu di install ga?

---

di folder instalasi espt pph pasal 21 memang tidak disediakan dotnet sama access database engine nya. Biasanya sudah terinstal bersamaan dengan OSnya. Tapi kadang ada kendala karena belum update dsb. Jadi sepanjang WP gak ada kendala apa apa saat menjalankan aplikasi esptnya, gaperlu unduh dan install access database engine. Biasanya muncul kendalanya cukup jelas juga diminta untuk install access database engine 2007, jadi tinggal unduh dan install sesuai error yang muncul. Boleh juga kalo WPnya
