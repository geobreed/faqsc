Selamat siang saya ingin bertanya saat saya membuat csv PPN-DM Muncul kode eror bu, bagaimana ya bu ? ini bagaimana ya mas mba?

---

1.Klik Start Menu, pada kolom Search programs and files ketik CMD kemudian klik kanan Run As Administrator, akan muncul jendela Command Prompt. 2.Kemudian jika menggunakan windows 32 bit, pada Jendela Command Prompt ketik cd C:\Program Files\DJP\eSPT PPN 1111 DM\dll kemudian tekan ENTER, tetapi jika menggunakan windows 64 bit, pada Jendela Command Prompt ketik cd C:\Program Files(x86)\DJP\eSPT PPN 1111 DM\dll kemudian tekan ENTER. 3.Setelah itu, ketik regsvr32 capicom.dll kemudian tekan ENTER
