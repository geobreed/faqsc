get data di prepopulated muncul etax api 10001, ada null pointer, sama nama lawan transaksi terisi nama pt sendiri.

---

etax api 10001 biasanya ada karakter yag tidak standar. jika tidak bisa melalui get data, coba input manual saja. untuk null pointer coba input ulang sertel. dan untuk nama lawan transaksi sama dengan napma pt sendiri karena uploader belum aktif. silakan nyalakan uploadernya dulu
