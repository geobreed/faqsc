#9Q : kak mau tanya terkait pelaporan spt ppn melalui web efaktur . wp akan melampirkan file setelah file di pilih dan klik lapor, loading tetapi tidak berhasil lapor. tidak ada keterangan error dan hanya circle loading dan tidak ter upload / terlapor. solusinya bagaimana ya kak? terima kasih

---

#9A: ini dari kemarin sih ada beberapa WP kaya gini dan belum ada info error dari efaktur. dicoba secara berkala dulu aja pake incognito/private window, c3l, ganti browser atau koneksi. bisa juga coba pasang ulang sertel di browsernya. pastikan laman efakturnya tidak di bookmark. solusi lain bisa coba hapus SPT nya terus posting ulang.
