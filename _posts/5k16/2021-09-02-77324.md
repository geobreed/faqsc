db efaktur corrupt apakah harus minta data ke kpp?

---

bisa disarankan pakai db dari folder back up dulu. tutup aplikasi efaktur, masuk ke folder back up, pilih tgl sehari sebelum db nya error, extract db nya kemudian gunakan untuk mengganti folder db di folder utama efaktur baru kiemudian buka lg aplikasi efakturnya. kalau folder back upnya kosong, minta data PK ke kpp
