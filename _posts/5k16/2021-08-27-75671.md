error etax api 10004 faktur normal yang akan diganti tidak ditemukan, sebelumnya WP ada migrasi database karena ganti OS lalu ada FP Batal yang hilang. WP ingin impor ulang lalu muncul notifikasi tersebut. Apakah karena statusnya FP Batal ya Mas/Mbak? tapi di db gak muncul katanya.

---

FP Normal > dibuatkan FP Pengganti > lalu FP Pengganti dibatalkan, sudah berhasil dan ada di sistem DJP. Tapi di desktop PKP tidak muncul krna ada kesalahan saat migrasi db. Setelah disarankan filter masih tidak muncul, maka diarahkan untuk ambil db di folder backup yg terupdate, ekstrak, copy and replace ke folder efaktur menggantikan yg lama. Jika ada data PK yg hilang, bisa minta KPP. PM bisa diinput ulang.
