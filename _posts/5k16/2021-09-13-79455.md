kenapa di folder single installer espt 21 tidak ada dotnet dan access database engine?

---

Kalau WP unduh single installer dari pajak.go.id memang isinya hanya folder debug saja, langkah selanjutnya di folder debug --> e-SPT package --> klik “Next”. Jika instalasi sudah selesai, maka akan muncul tampilan installation complete. Untuk .NET Framework dan access database engine bukan aplikasi DJP silakan WP bisa unduh sendiri dari mesin pencarian.
