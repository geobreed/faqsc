saya ada mao buka aplikasi e-SPT PPH OP tahun 2016 dan 2017 namun muncul notifikasi convertion from type 'DBNull' to type 'Integer' is not valid, ini kenapa ya mas/mba?

---

Solusinya hanya buka databasenya kemudian isi periode pembukuan. error ini muncul ketika setelah patch ke 1.6.1 kolom periode pembukuan di dalam file database jadi kosong harus isi manual. tapi untuk membuka database butuh password, bisa konfirmasi ke KPP untuk membuka databasenya.
