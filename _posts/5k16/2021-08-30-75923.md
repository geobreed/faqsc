efaktur desktop ETAX-10004

---

Mohon maaf atas ketidaknyamanannya. Apakah saat ini masih terkendala? Untuk Error ETAX-10004, silakan klik get data kembali untuk mengulangi proses request ke server DJP karena kemungkinan koneksi terputus. Coba juga untuk stop uploader dan nyalakan kembali. Sudah coba rekam manual, Kak? Kakak juga dapat coba cara berikut: 1. Pastikan koneksi internet stabil. 2. Coba matikan Antivirus dan/atau Firewall sebelum buka aplikasi efaktur kembali. 3. Jika menggunakan proxy, pastikan aplikasi efak
