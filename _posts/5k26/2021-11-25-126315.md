coba pakai cara ini dulu ya Silakan coba ikuti langkah-langkah berikut: 1. Save file e-Form terlebih dahulu, kemudian tutup. 2. Buka Internet Explorer, kilk Options (icon gir di pojok kanan atas). 3. Di bagian Network and Internet, cari Internet Options. Setelah terbuka, klik Advanced di pojok kanan atas. 4. Di bagian Setting, scroll ke bawah cari bagian Security (gambar gembok kuning). Kemudian, centang semua SSL dan TLS. 5. Buka kembali file eform (.pdf). Coba submit SPT kembali. 6. J
