unhandled exception ini belum kita ketahui pasti sebabnya apa. bisa dicoba cara ini dulu. Error unhandled exception: 1. solusi paling gampang dulu: RUN AS ADMINISTRATOR dengan klik kanan exenya; atau 2. Coba instal ulang crystal report; atau 3. Pakai cara teknis tapi agak ribet: a. Pertama cek Komputer/Laptop, menggunakan windows 32 bit atau windows 64 bit, karena caranya akan berbeda nantinya. b. Masuk ke Command Prompt. Caranya pada kolom Search programs and files ketik CMD kemudian tek
