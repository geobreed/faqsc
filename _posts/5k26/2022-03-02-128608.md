wp sudah merekam NSFP sesuai jatah yang diterima dari enofa tapi ketika akan rekam FP baru, kenapa kolom nomor seri nya kosong ya? jadi harus rekam manual terus

---

sepertinya ada masalah dengan DB nya. silakan coba arahkan wp untuk membuat DB baru, kemudian rekam NSFP nya di DB bau dan buat fakturnya juga dari db baru. nanti untk lapornya karena sudah via web efaktur sehausnya tidak ada masalah. apabila tetap ingin menggunakan db lama, mungkin diarahkan membawa file db ke KPP untuk dibantu via lasis.
