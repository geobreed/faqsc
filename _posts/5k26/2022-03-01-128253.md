#24Q Twitter: https://twitter.com/U_Dayn/status/1498478927190265859 izin bertanya Mas/Mbak kalo ada kode error SSP tidak valid yg diinput bukti PBK, apakah dicek saat pelaporan di web efaktur dl atau seharusnya tetap valid di efaktur desktop?

---

#24A: Untuk penginputan SSP/Pbk cukup dilakukan di Web eFaktur saja ya. Jika tetap ingin input pun pastikan penulisan nomor Bukti Pbk sudah sesuai dan telah mencentang "PBK" di samping kolom NTPN.
