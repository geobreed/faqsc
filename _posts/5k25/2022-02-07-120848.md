#28Q : saat melakukan impor pada ebupot unifikasi untuk PPh Pasal 23 muncul kode ERRV0000 Terdapat draft Bukti Potong yang gagal di-validasi ! Silahkan perbaiki file excel ini kenapa ya Mas/Mba?

---

Palingan cuman arahkan wp memastikan isian dan format di file excelnya sudsh benar dan sesuai ran, coba impor lagi berkala, apabila masih tetap tidak bisa, boleh diarahkan bawa file excelnya ke kpp untuk diteliti dan dicoba kembali Klik aksi Lihat. Di menu ‘Lihat’ akan terlihat error-nya di bagian mana dalam file excel yang diimpor beserta alasannya. Sesuai alasan tsb, silakan diperbaiki dan impor ulang.
