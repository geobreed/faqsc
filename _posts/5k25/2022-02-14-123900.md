ini sarankan run as admin sm instal ulang, atau ad yg lain kah mas/mbak?

---

Kendala seperti ini biasanya terjadi untuk komputer yg RAM tersisa tinggal sedikit . Karena default efaktur 3.1 RAM yg digunakan cukup besar, untuk itu disarankan WP untuk setting memconfignya untuk mensetting besaran RAM yg digunakan ketika efaktur dijalankan dari 10000 mb ke 2000 mb . Silakan arahkan WP untuk : 1. buka file bernama "mem_config.bat" pada folder efaktur 3.1 2. pada akhir kalimat "Besar Maxiumum Memory yang digunakan dalam MegaByte? [Kosongkan untuk AutoSet]" tulis angka "2000"
