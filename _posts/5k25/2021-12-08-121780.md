sudah menginput NIK dari lawan transaksi OP pada ebupot. muncul error berikut: Error membaca cell NPWP, Conversion from type 'DBNull' to type 'String' is not valid.; NPWP Wajib Diisi

---

Error 'DBNull' to type 'String' untuk metode impor excel, harus mengisi semua data, dengan nama kota/kab dan provinsi sesuai nama pada sheet referensi pada excel. pastikan format excel tidak diganti dan mengikuti petunjuk sesuai di command cell. Pastikan wp pake format excel yg terbaru WP dapat mengunduh ulang skema impor tersebut pada laman impor di sebelah kiri. pastikan format excel tidak diganti, sarankan untuk tidak copas dari excel sebelumnya agar format cell tidak berubah
