kendala runtime 339 saat ingin ekspor data di espt Badan 1771 2010, izin tanya mas/mba klo kendala ini solusinya gimana ya?

---

Setelah digali , keterangan lengkap error " Run-time error '339' Component Comdlg32.ocx or one of its dependencies Not Correctly Registered : a file is missing or invalid." Hal tsb karena file Comdlg32.ocx belum tersedia di perangkat dia/udah corrupt dan espt badan membutuhkan file Comdlg32.ocx tsb ketika akan melakukan ekspo/impor data , mba . Solusi 1.Download file comdlg32.ocx 2. Jika Anda pengguna Windows 32 bit, simpan file tersebut ke direktori c:\windows\system32 Jika Anda pengguna
