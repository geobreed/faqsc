etax web 50008

---

ETAX-WEB 50008 ada beberapa kemungkinan 1. file tidak sesuai ketentuan (format dan size lampiran) 2. terjadi kesalahan saat menyimpan file di server 3. terjadi kesalahan saat proses generate bps Pastikan filenya pdf dan ukurannya ya maks 5mb. Pastikan koneksi stabil. Boleh dicoba juga c3l. Info dari orang TIK sedang ada masalah generate BPS, jadi bisa coba berkala.
