Etax 10003 input database tidak berhasi;

---

enyebab: [1] Koneksi antara server dan client terputus (Network Database). [2] a. Merekam atau mengubah data user, referensi lawan transaksi, refer ensi barang/jasa di database selain ETaxInvoice; b. Mengupdate profil PKP di database selain ETaxInvoce. [3] Ada isian data faktur yang lebih dari 255 karakter baik untuk nama lawan transaksi, alamat lawan transaksi, nama barang ataupun refe rensi. Solusi : [1] Pastikan PC Server dan Client terkoneksi dengan baik. Tutup kembali aplikasi e-f
