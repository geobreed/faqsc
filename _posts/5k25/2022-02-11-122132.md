mau tanya, efaktur untuk databasenya jika pakai network apa busa dari beda lokasi? kalau bisa bagimana caranya ya?

---

untuk network database dengan skema server client bisa dijalankan sepanjang PC Server dapat terhubung dengan PC Client melalui jaringan. Komputer client tersebut harus terhubung melalui jaringan baik melalui kabel jaringan (LAN) atau menggunakan jaringan nirkabel (wifi). Terkait teknis cara pengaturannya silakan arahkan wp ke menu help efaktur dekstop dan pilih Konfigurasi Client Server Aplikasi eFaktur, Dalam menu help tersebut sudah ada requirement dan konfigurasi yang harus dipenuhi jika in
