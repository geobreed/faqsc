Kak, nama di Dokumen lampiran LK SPT Tahunankan, pakai nama CSV lalu ditambah LK ya.

---

Pemberian nama file lampiran SPT selain untuk SPT PPh Badan (1771) dan SPT Tahunan OP (1770) yang berekstensi pdf harus sama dengan nama file csv, contoh jika file csvnya bernama 0123456789520010101201200F1232040111.csv, maka file pdfnya harus diberi nama 0123456789520010101201200F1232040111.pdf, sedangkan untuk SPT 1771 dan 1770 nama file pdf harus ada kode sesuai keterangan disamping tombol "Pilih File...". WP mau upload csv spt tahunan tapi di penamaan lampiran Lapkeunya pake kode CSV PPN
