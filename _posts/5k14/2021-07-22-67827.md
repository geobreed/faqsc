kendala ebupot error object object saat impor excel

---

Info dari TIK, untuk pertanyaan terkait error object object pada saat impor, itu biasanya ada format cell yg tidak sesuai excel asli dari DJP. Format cell-nya berubah. Sering terjadinya karena WP copas data termasuk formatnya dari tempat lain, yg mengubah format asli excel impor dari DJP. Minta WP nya memperbaiki. Jadi coba berkala dulu, sampai saat ini belum ada solusi lanjutannya.
