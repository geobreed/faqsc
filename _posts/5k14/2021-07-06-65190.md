wp lupa username dan password efaktur. Apakah langkah di bawah ini juga bisa mereset kembali usernamenya?

---

secara garis besar: pastikan sertel masi berlaku. 1. step 1, permintaan reset password dari aplikasi efaktur. klik lupa password sebelum login efaktur desktop. (lebih jelas, baca help. bagian lupa password admin utama) 2. step 2, reset aplikasi efaktur desktop melalui web enofa. (lebih jelas, cek di efaktur.pajak.go.id, sebelum login, di bagian nomor 5 ada manual instalasi sertel, download filenya. bagian romawi VI ada petunjut reset registrasi aplikasi efaktur, ikuti langkahnya 3.
