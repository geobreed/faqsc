#6Q: (Email) Saya mau tanya kenapa E-Faktur saya tidak bisa Log-in saat mau dibuka hanya loading saja dan muncul tulisan "Java Runtime Eviroment Not Found". Itu maksudnya apa ya? Padahal Senin sore saya masih bisa log-in dan input faktur pajak.

---

#6A: error java runtime environment biasanya karna versi aplikasi e-Faktur yang dipake beda dengan versi OS (32bit atau 64bit) di PC nya jad coba dipastikan dulu itu yu. kalo udah sesuai tapi masih begitu kemungkinan memang javanya ilang atau rusak. sarankan install ulang java runtime environment nya. opsi terakhir install ulang aplikasi efaktur nya, tapi pastiin backup db dulu buat dipindahin ke aplikasi yg baru.
