#142Q Perusahaan tempat saya bekerja kehilangan FP Keluaran dan telah request ke KPP. Data sudah dikirimkan lewat email dalam bentuk folder Zip. mohon ifno bagaimana saya upload kembali data dari KPP ke Efaktur?

---

#142A: file zip nya diekstrak dulu nanti isinya data faktur keluaran dalam bentuk csv. setelah itu tinggal impor ke faktur keluaran di efaktur dekstop. caranya klik menu faktur > Faktur keluaran > Impor > Open file > Pilih data file yg udah ada > Open > Proses panduan lengkapnya bisa di lihat di menu help efakturnya ya
