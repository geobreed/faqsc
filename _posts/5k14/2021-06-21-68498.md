WP melakukan reset client tapi Kode Aktivasi Desktop tidak berubah bagaimana ya?

---

Pandu wp reset aplikasi client. Setiap kali klik reset aplikasi client langsung kelogout enofanya, belum sampai pada pengisian kode aktivasi ama password enofa, belum sampai juga ada informasi kalo aplikasi efaktur desktop berhasil direset, udah cek diprofil enofa(status, kode aktivasi masih sama) dan di admin enofa memang belum reset. Belum ada informasi errror, bisa c3l dulu dan saranin hapus sertel lalu install lagi sertel di browser
