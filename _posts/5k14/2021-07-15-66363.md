WP sudah pakai eSPT 1771 ver 2010 (1.2) tapi karena WP mau buat SPT thn 2021, disarankan kepada WP utk install yg ver. 2009. Saat WP install yg 2009, installer nya minta “modify”. Apakah dilanjut saja atau harus install di dua perangkat berbeda atau bagaimana ya seharusnya? Terima kasih.

---

instalasi eSPT 2009 memang seharusnya diinstal diperangkat yg sama dgn eSPT 2010 karena db yg akan digunakan kedua aplikasi itu sama. Jika muncul modify itu kemungkinan aplikasi yg versi 2009 sudah terinstall di perangkat WP. Arahkan WP untuk memastikan apakah ada folder dengan nama "eSPT 1771" (versi 1.0), jika sudah ada WP dapat menggunakan aplikasi tersebut. Jadi, baiknya digali dulu saja mas, di folder aplikasi DJP (biasanya ada di C:\Program Files (x86)\DJP) apakah ada 2 folder aplikasi eSP
