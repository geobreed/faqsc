Kalau WP input dokumen lain PM untuk transaksi JKP/BKPTB, muncul error ETAX API 50033 sama ETAX 30031 itu kenapa ya mas mba?

---

1. pastikan NTPN yang diinput sudah benar 2. pastikan kode KPP nya benar (kode KPP administrasi) 3. pastikan NPWP yg diinput saat bikin id billing sudah benar 00.000.000.0.kode kpp.000 (checklis NPWP lain) 4. pastikan pembayaran memang sudah masuk di sistem
