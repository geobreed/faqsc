error web 50008 web efaktur pas mo lapor. masih 3cl dan coba berkala? atay ada update teranyar?

---

ETAX-WEB 50008 ada beberapa kemungkinan 1. file tidak sesuai ketentuan (format dan size lampiran) 2. terjadi kesalahan saat menyimpan file di server 3. terjadi kesalahan saat proses generate bps Pastiin filenya pdf dan ukurannya ya maks 5mb. Pastiin koneksi stabil. boleh dicoba juga c3l
