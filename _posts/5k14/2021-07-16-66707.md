wp sudah berhasil perpanjang sertel tapi pas dimasukin muncul error etax 30023 : file certi tdk dapat dipakai. penyebabnya apa ya?

---

di katalog error kemungkinan karna file sertelnya salah. Pastikan sertel masih berlaku dan coba donlot ulang. Format nama sertel dibiarkan aja jangan diubah ubah. Boleh disimpan di folder khusus supaya mudah dicari
