input dok lain kehapus, mau input lagi tulisannya nomor sudah terdaftar. Bagaimana?

---

setelah digali dok lain pm sudah approval sukses dan dibatalkan oleh wp di efakturnya, saat input ulang muncul etax api 50003. pastikan dulu di web efaktur sudah masuk atau belum untuk pm-nya. untuk memunculkan kembali dok lain pm-nya konfirmasi kpp.
