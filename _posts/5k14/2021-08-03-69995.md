Mau bertanya apakah MAP code berubah utk pembayaran Bunga Obligasi utk WPLN sebelum nya MAP Code 411128-401 menjadi 411127-102 ?

---

Untuk pembayaran bunga kepada WPLN terutangnya PPh pasal 26, maka seharusnya MAP KJS 411127-102
