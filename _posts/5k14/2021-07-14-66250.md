Unhandled exception has occured in your application. IF you click continue, the application will ignore this error and attempt to continue. If you click quit, the application will close immediately dan error lanjutan 'No data exists for the row / column'

---

Error ini muncul karena belum terinstalnya Crystal Report Basic Runtime for visual studio 2008 atau sudah terinstal tetapi rusak (tidak bekerja). Solusinya Instal Crystal Report. Jika sudah terinstal, uninstal terlebih dahulu kemudian instal kembali. Setelah instal atau reinstal crystal report restart Komputer atau laptop.
