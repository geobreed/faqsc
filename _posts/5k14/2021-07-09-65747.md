saya ingin menanyakan mengenai aplikasi PPH Final pasal 4 (2), kemari kan saya sudah insall aplikasi terbaru yg 2.1. Nah saat saya mau cetal Daftar Bukti Pemotongan itu saat dicetak pdf kosong tidak ada isinya

---

Mau cetak Bukti potong untuk diserahakn ke WP PP 23, memang tidak bisa cetak bukti potong PP 23, karena pemotong bukan menyerahkan bukti potong, tapi menyerahkan bukti bayar, kalau ikut insentif WP nya, berarti kode billing insentif yang diserahkan, bukan bukti potongnya. Untuk daftar bukti potongnya, disarankan WP untuk restart aplikasi menggunakan Run as Administrator, coba install pacth update di pajak.go.id, kalau belum bisa coba hub. AR di kpp untuk minta pacth bug cetak bupot.
