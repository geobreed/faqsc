mas/mbak, wp baru saja edit user di efaktur. Pas udah disimpan dia gak bisa liat pk pm yg lama padahal udah di cek menang cuma 1 user admin yg ada. Itu gimana ya mas/mbak?

---

Kemungkinan WP menambah user, kemudian menghapus user lama sehingga PK dan PM lama tidak terbaca. Disarankan untuk mengganti db dengan db backup sebelum WP ganti user.
