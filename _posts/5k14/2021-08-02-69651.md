Error aplikasi AEOI format file tidak diizinkan, solusinya?

---

Di katalog error eoi tidak ada error tsb, coba pastikan terlebih dulu file yang diupload sudah sesuai dengan format xml yg dari pajak.go.id dan dilakukan enkripsi (pastikan langkah-langkahnya di user manual EOI). Kalau masih tidak bisa arahkan WP ke email eoi@pajak.go.id atau ke KPP.
