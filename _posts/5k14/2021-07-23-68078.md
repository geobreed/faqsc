mas/mbak kalo wp mau lapor SPT PPN 1111 DM terus muncul error tersebut, karena apa ya? Error di e-SPT PPN 1111 DM: data type mismatch in criteria expression

---

Cek setting regionalnya pastikan udah Indonesia, Cek pada bagian Formats - Long Date, pastikan dalam format "ddMMMMYYYY". Apabila belum berhasil, silahkan restart PC/laptop dan coba install ulang aplikasi.
