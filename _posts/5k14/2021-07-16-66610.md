Saya sdh install espt pph badan 2009 dan 2010 di satu komputer tp yg terjadi aplikasi versi 2009 ke update jd 2010. Bagaimana ya min?

---

ketika download file installer espt tahunan badan, ada 4 file yang akan kita jalankan. ada file 1, 2,3, dan installer espt dollar. nah udah diikutin belum cara instalasi nya kak? ada juga di file tsb tata cara instalasi, yaitu: 1. Install terlebih dulu file "1.exe" (Installer e-SPT PPh Badan Tahun 2009) 2. Install file "2.msi" (File ini berisi update ke e-SPT PPh Badan Tahun 2010) 3. Install file "3.exe" (File ini berisi patch e-SPT PPh Badan Tahun 2010) 4. Lakukan proses instalasi sesuai
