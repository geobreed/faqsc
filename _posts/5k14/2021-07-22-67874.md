untuk input transaksi, bisa langsung import dari csv tanpa harus input per transaksi di aplikasi efaktur?

---

Bisa impor pake csv, caranya, wp suruh input dulu satu faktur pajak keluarannya di efaktur, setelah itu lakukan ekspor datanya melalui menu faktur-pajak keluaran-export. Hasil csv yang dia export tadi bisa dijadikan format untuk import data fp keluaran ke aplikasinya. Cara importnya melalui menu faktur-pajak keluaran-import. Sama dengan impor csv faktur pm. Hanya beda menu saja, pilih faktur-pajak masukan.
