(Email) Saya wajib pajak, pada kolom profil data diri tidak valid. Namun pada saat saya mau melengkapinya, bagian kolom NIK tidak dapat diisi, sehingga proses berikutnya tidak bisa dilakukan, disamping itu saya juga mau menambahkan anggota keluarga. -------------- Mas/mbak ini kenapa ya?

---

ada tangkapan layarnya kah? ketika status data utama tidak valid (ada notif perlu dimutakhirkan), seharusnya kolom NIK, nama, TTL terbuka semua kolomnya, silahkan diisi dengan benar. jika masih tidak valid atau muncul error biasanya karena datanya gak cocok dengan data dukcapil. kalo aku coba, ketika data utama belum berhasil valid, bisa kok klik tab berikutnya misalnya data lainnya, dan bisa isi dan ubah profil. untuk pengisian anggota keluarga juga dapat dilakukan dan pastikan datanya lengkap diisi ya, nik, nama dsb. jika kolom nik kosong tapi tidak dapat diisi di bagian profil utama, coba sarankan c3l, private window dsb dulu, harusnya kolom tsb bisa diisi. jika sudah valid, maka kolom data utama terkunci.
