---

sepertinya db efakturnya crash. silakan hapus folder db, kemudian ekstrak db cadangan dr folder backup, ambil tanggal yg paling mendekati tgl hari ini, silakan diektrak sampai muncul folder db, pindahkan folder db hasil ekstrakan ini ke folder utama efaktur menggantikan db yg td dihapus. apabila tidak ada data di folder back up atau tidak ada data back up sama sekali, reset dan register ulang efaktur, nanti minta data PK ke kpp
