mas mba wp email begini: apakah DJP memiliki API (Application Programming Interface) yang dapat kami gunakan untuk melakukan pengecekan validitas NPWP secara otomatis?, apakah memang ada API yang dimaksud dr DJP?

---

kalau API yang dimaksud wp ini adalah aplikasi untuk pengecekan validitas NPWP secara mandiri belum ada sa, paling bisa cek KSWP aja di DJP Online itupun untuk wp ybs bukan untuk lawan traksaksinya karena harus log in djp online ybs. atau dari ereg/ceknpwp tapi harus input nik dan juga KK. kalau mau pengecekan NPWP saat ini bisa cek ke KPP atau kring pajak untuk layanan validasi NPWP.
