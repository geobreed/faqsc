Saat mau buka efaktur terdapat notifikasi "tidak dapat menjalan d, java virtual machine starts failed",

---

Bisa coba-coba dulu cara ini, coba cek java yg terinstall di pc wp, kalau sudah ada coba install ulang minimal versi 7. Kalau sudah install dan terupdate javanya dan masih keluar info error itu kemungkinan aplikasi efakturnya corrupt, coba sarankan wp install ulang aplikasinya trus db lamanya di masukan ke aplikasi yg baru diinstall td.
