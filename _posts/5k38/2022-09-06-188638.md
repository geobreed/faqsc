jika melihat notifikasinya, "tidak dapat melakukan backup","There is not enough space on the disk", sepertinya terkait sisa memori di drive nya, jadi ketika membuka eFaktur, aplikasi akan meng-create file backup db, nah saat proses ini, tidak cukup memori drive nya. coba arahkan untuk melakukan pengecekan sisa space drive nya, atau cek di dalam folder backup ada brp banyak file backup, biasanya folder ini yg banyak makan space memory. kalo memang banyak, bisa dipindahkan sebagian ke tempat penyimpanan lain
