di folder efaktur gak ada folder backup, gimana?

---

Folder Backup di folder efaktur tidak ada kemungkinan karena file ETaxInvoiceUpd direname, sebenernya bisa dimunculin dengan mengembalikan nama defaultnya, cuman memang disarankan untuk diganti atau tambahi old biar enteng perangkatnya, konsekuensinya memang harus back up manual folder db nya
