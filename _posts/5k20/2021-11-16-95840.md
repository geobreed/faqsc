Penamaan File ebupot gimana ya

---

Aturan penamaan file adalah diawali dengan <15 digit NPWP>.xls Contoh: untuk WP dengan NPWP 123456789012345, nama file dapat berupa: 123456789012345.xls 123456789012345_1.xls 123456789012345 20171011 1.xls
