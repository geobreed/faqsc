#65Q: Ini saya ada PIB tahun 2018 mau saya ubah tapi tidak bisa, kenapa ya? etax api 50009 dokumen yg akan diubah tdk ditemukan

---

#65A pm ya mbaa mau ubah nominal, muncul ETAX API 50009 dan ETAX 20049 ubah DPP dan PPN menjadi 0, lalu input PIB lagi karena ini adalah pembetulan dari SPT 2018, maka silakan buat SPT pembetulan menggunakan efaktur 3.0 > buat CSV > laporkan melalui efiling djponline
