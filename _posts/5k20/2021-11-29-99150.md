input dokumen lain pajak masukan reject etax api 50030

---

Untuk ETAX API 50030 notifnya data tidak ditemukan. Pastikan tata cara penginputan di dokumen lainnya sudah sesuai. Untuk tata cara penginputan nomor dokumen tergantung dokumen impor WP, Impor PIB (bc2.0), bc 2.8: No.pendaftaran#ntpn. Pastikan nomor dokumen, tanggal NTPN, sudah benar. Jika masih tidak ditemukan arahkan WP untuk konfirmasi ke BC terlebih dahulu melalui customer.beacukai.go.id atau telp. ke help desk Bravo Beacukai 1500225 atau bisa coba konfirmasi ke KPP juga.
