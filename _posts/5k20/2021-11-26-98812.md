Saat akan input cbcr, cbc valid, tapi kertas kerja muncul error: cvc-complex-type.2.4.a: Invalid content was found starting with element 'Report'. One of '{report}' is expected. Ini kenapa ya?

---

Beri nama file XML yang telah divalidasi sesuai petunjuk penamaan file XML CbCR. Kemudian arahkan wp untuk mengakses https://pajak.go.id/id/cbcr untuk melihat petunjuk pengisian file XML secara lebih detail. Terkait pembaharuan di sistem e-cbcr, belum ada info akan berlangsung sampai kapan. Arahkan wp untuk unduh kertas kerja di https://pajak.go.id/id/cbcr.
