error efaktur. etax 10006 Tidak dapat menstart database sebagai network server

---

Dijelaskan sesuai katalog. terjadi ketika sistem efaktur wp menggunakan sistem server client. penyebabnya biasanya karena Port yang digunakan untuk koneksi database belum dibuka. solusinya Pastikan port yang dipakai untuk koneksi database telah dibuka. jika masih ada trouble coba cek kembali konfigurasi server client nya. untuk bantuan bisa cek lengkapnya di menu help aplikasi efaktur
