Izin tanya Mas/Mba. WP input pembetulan dok lain pajak keluaran untuk MP 04/2019, dengan dokumen PEB. Saat input no dokumen berupa nomor pendaftaran PEB, muncul notifikasi: ETAX 20049 Dokumen lain gagal diupdate dan ETAX API 50042 Format penulisan PEB tidak valid, seharusnya (NOPEB#NOAJU). Ketika input kembali dengan ketentuan format (NOPEB#NOAJU), muncul notif error ETAX 20049 dan ETAX API 50009. Untuk solusinya bagaimana ya, Mas/Mba? Terima kasih.

---

errro ETAX 20049 dan ETAX API 50009 ini muncul karena wp mau merubah data PEBnya. coba input ulang saja PEB nya dengan ketentuan yang benar no peb#no aju. untuk peb yang salah coba ganti dppnya jadi 0 dan ganti jadi penyerahan dalam negeri.
