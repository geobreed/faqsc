WP mengalami kendala login di aplikasi espt PPh 21 versi terbaru. Ketika login dan memilih database muncul notif :"the operating system' is not presently configured to run this aplication" sebelumnya sudah install ulang dan masih muncul kendala yg sama

---

Kemungkinan operating system komputer yg digunakan WP bermasalah dan tidak kompatibel dalam menjalan aplikasi espt 21. Bisa disarankan install aplikasi espt 21 di komputer yg berbeda, lalu db nya pindahkan ke aplikasi espt 21 yg baru diinstall di komputer yg berbeda tersebut.
