ETAX-10006. Sudah saya periksa dengan firewall dan port dan memeriksa dengan Troubleshooting/Help. Namun, belum dapat terselesaikan.

---

Dijelaskan sesuai katalog. terjadi ketika sistem efaktur wp menggunakan sistem server client. penyebabnya biasanya karena Port yang digunakan untuk koneksi database belum dibuka. solusinya Pastikan port yang dipakai untuk koneksi database telah dibuka. jika masih ada trouble coba cek kembali konfigurasi server client nya. coba sarankan wp back up db dulu, konfigurasikan ulang saja sesuai menu help.
