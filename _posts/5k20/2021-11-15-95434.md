WP install ulang efaktur karena terkena virus. Pas udah instal ulang diminta kode aktivasi. Itu kode aktivasinya tinggal cek aja di efatur kan ya? Dia nanya permintaan ulang kode aktivasi

---

Komputer wp terkena virus, wp lakukan reset aplikasi client. Ketika akan masuk lagi ke efaktur setelah install ulang, akan diminta registrasi kembali, jadi input kode aktivasi efaktur desktop (yg panjang), yg muncul setelah reset aplikasi client.
