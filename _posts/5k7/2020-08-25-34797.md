DB Espt Final PPh final 1.0, apakah ini kemungkinan connect db yg salah ?

---

iya bisa jadi. kemungkinan wp salah copy db. Penyimpanan db ada yg di virtual store ada yg tidak. Tergantung setting user account controlnya. Kayaknya db yg sebelumnya sudah ada isinya tersimpan di virtual store. Bisa disarankan ke wp untuk coba copy db nya dari virtual store dulu ya, baru nanti setting odbc untuk memasukan db yg diambil dr VS tadi. Note : espt pph 4(2) versi 1.0 dan pph 23 pake setting odbc untuk memasukan db.
