Saya sudah cek eform, memang bentuk file nya xfdl, dan menu print nya tidak nyala. untuk solusi print nya di screenshoot lalu print ya mas/mbak?

---

Caranya bisa buka lagi file eformnya yang dalam bentuk XDFL Document (ekstensi .xdfl) lalu klik halaman selanjutnya sampai halaman submit (tapi jangan disubmit lagi karena tujuannya hanya cetak) kemudian ada pilihan untuk cetak/print. klo file xdfl nya sudah ga ada coba konfirmasi ke KPP.
