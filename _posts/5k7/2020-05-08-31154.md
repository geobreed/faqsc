kalau wp udah berhasil memasukkan ntpn ke efaktur tp pas create csv selalu muncul etax-20044.

---

ETAX-20044 : SPT Kurang Bayar Penyebab : Membuat File CSV untuk SPT Kurang Bayar tanpa merekam SSP terlebih dahulu Solusi : Dalam hal SPT Kurang Bayar, rekam terlebih dahulu SSP nya 1. Coba pastikan lagi mbak untuk data pembayaran sudah benar. 2. Jika sudah benar, minta WP untuk hapus SPT yang sudah dibuat kemudian posting kembali. Kemudian isi kembali NTPN nya. Kemudian dilanjutkan dengan create CSV.
