apakah web efaktur sedang bermasalah? saya sudah isi lengkap SPT PPN saya tetapi muncul 'SPT tidak Valid harap periksa kembali isian anda'

---

Silakan dicoba pakai cara berikut : 1. Melakukan Clear Cache 2. Menghapus Semua Sertel di Browser 3. Unduh ulang sertel melalui enofa Online 4. Tutup Browser 5. Impor Sertel kembali ke Browser yang digunakan 6. Akses kembali menu efaktur Web. Buka spt, (Untukc case kalau ada kompen lb dr masa sebelumnya yg tidak masuk spt ke induk) maka ke lampiran 1111AB, 0 kan dulu yg kompensasi kelebihan dr masa pajak sblmnya, simpan trus input lg nilai kompensasi LB, simpan. Kemudian cek lg spt induk
