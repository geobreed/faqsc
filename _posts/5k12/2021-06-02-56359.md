Saya sampai saat ini masih terkendala login, kondisi hardisknya juga tdk penuh min, tp sampai skrg masih belum bisa login, keterangannya seperti itu: Update Database Gagal - An SQL data change is not permitted for a read-only connection, user, or database. dan Tidak dapat melakukan backup java.io.IOException: There is not enough space on the disk. kemarin ketika mau upload pun jg sempat muncul error etax-10002. itu kenapa ya?

---

notif pertama itu kayak yg biasanya muncul waktu orang gagal patch update, coba untuk login tanpa koneksi internet notif kedua ada kata2 space yg kurang, pastikan space yg masih tersedia sesuai dengan minimum requirement yg di minta oleh efaktur (coba liat di menu help efaktur, berapa GB nya) notif ketiga saat upload, pake solusi sesuai katalog error, kayaknya kaitan sama server client, selain itu coba juga cara ini : https://www.efakturespt.com/2016/11/solusi-e-faktur-error-etax-10002-tidak.h
