saya beberapa waktu yg lalu ada mengajukan permintaan data PKPM ke KPP palangkaraya dan sudah di berikan datanya oleh kpp tapi saat saya masukan ke aplikasi efkatur tidak isa. error ETAX 60002. solusinya bagaimana ya pak?

---

Error tsb muncul karena WP terlanjur mengekstrak file zip data PK tsb. Harusnya file data PK yg diperoleh dari KPP langsung diimpor ke dalam efaktur (tanpa diekstrak dulu).
