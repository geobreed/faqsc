Bagaimana caranya e-faktur bisa digunakan di dua komputer tanpa mereset data client dari komputer pusat?

---

ini maksudnya untuk buat kayak server-client gitukah? kalo client server ada tata caranya di menu help efaktur yak..kalo hanya untuk buka aplikasi efaktur di komputer lain bisa tinggal copy paste folder efaktur ke komputer lain. cuman data nya tidak update ya, jadi kalo misal ada input PK di komputer A. gak otomatis muncul di komputer B. bisa pake impor atau ekspor, atau tinggal copy paste lagi folder efaktur desktop nya.
